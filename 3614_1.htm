Manage Online Privacy Messages (for webHotel Sites)

This feature provides the ability to setup as many as four unique privacy messages to display during the booking process. Use the privacy (page area) components to add privacy messages to custom page layout for the Modify Guest Details page or the modify Reservation page. Define the online privacy messages as mandatory or not. Specify the online privacy text to display on the page or to display in a modal popup. The online privacy responses update in OPERA through the User Defined Fields (UDFs), this information is stored in the OPERA on the Reservation.

Add up to four Privacy Messages:

  • Communication Privacy
  • Consent Privacy
  • Terms & Conditions Privacy
  • General Privacy

Configure the Manage Online Privacy Process:

Step 1. Setup Reservation UDFs in OPERA to correspond to the privacy messages in Site Manager
Step 2. Import LOVs from OPERA
Step 3. Create the privacy messages in Site Manager
Step 4. Configure Online Privacy Options
Step 5. Configure the Page Layouts for to include the Privacy Messages using Components

Step 1. In OPERA Setup UDFs to correspond to the privacy messages in Site Manager

The OPERA Reservation UDF fields correspond to the Reservation UDF Fields in Site Manager. Mapping the Online Privacy Messages to UDFs in OPERA provides the ability to save the user's reply to the Reservation in OPERA.

If setup the UDFs are defined and screen painted on they display on the OPERA Reservation Screen.

  • In ORS > Sales Screen > Retrieve reservation > go to More Fields
  • In PMS Property > Update Reservation

Make sure the Privacy UDFs are active in OPERA.

To add these UDFs to the Reservation in OPERA Setup > Screen Design > List Of Values Maintenance to add these UDFs to the Reservation screen.

Step 2. To Import LOVs from OPERA

  1. In Site Manager go to the User Options tab on the top navigation.
  2. Then from the left navigation select the Import All LOV Types from OPERA
  3. Verify the Reservation UDFs import.

Reservation UDFs in Site Manager

If the UDFs are setup in OPERA and after the Import All Lov Types from OPERA to Site Manager the Privacy UDFs display in Manage Sites > Frontend Settings > OPERA LOV Settings > Reservation UDF.

Note. The UDF fields data type needs to be specified as Text.

There is nothing to configure here just confirm the successful import from OPERA.

  1. In Site Manager select Site Settings from the top navigation.
  2. Select Manage Sites from the left navigation.
  3. Go to the Site's Front End Settings > then select the OPERA LOV Settings tab > and the Reservation UDF tab to verify the UDFs imported successfully from OPERA.

Step 3. Create Online Privacy Messages in Site Manager

  1. In Site Manager go to the Content tab and select Manage Online Privacy from the left navigation.
  2. Select a Property Code from the Property Code drop down list.
  3. Select a Message Type:
  • Communication
  • Consent
  • Terms & Conditions
  • General
    1. Message Name. Enter a Name for this message.
    2. Select the Create Message

Step 4. Configure Online Privacy Options

Select the Online Privacy Options & Configuration tab to configure the online privacy messages.

Mandatory. Turn on or off to make the privacy message mandatory. If this is set to On, the user is required to select Accept or Not Accept.

Modal. Turn on to enable the privacy message to appear on a popup modal. Turn off and the privacy message displays on the page. Turn On to display the "Privacy Details" link and if the user clicks on the link, the modal popup displays the Privacy Message. Turn Off is to show the Privacy Message directly in the component on the page layout and not in a popup. The default value is Off.

Privacy Message. The privacy drop down menu is populated by the privacy messages that are created on the previous step. Content > Manage Online Privacy

UDF: Select the UDF value these UDF values come from the Manage Sites > Front End Settings > OPERA LOV settings > Reservation UDF.

Step 5. Configure the Page Layouts to include the Privacy Messages using Components

The privacy messages can display on either of these pages:

  • Modify Guest Details page: Booking Path Pages > guest_info_modify_reservations.jsp
  • Modify Reservation page: Booking Path Pages > modify_reservation.jsp
    1. In Site Manager go to the Content tab and select Sections & Pages from the left navigation.
    2. go to Content > Sections & Pages > Booking Path Pages > Select Edit next to either guest_info_modify_reservations.jsp or modify_reservation.jsp
    3. Use the Privacy 1-4 system components to add the privacy messages to page layout.
    4. Schedule a new layout or edit a current layout using the Privacy components Save and Apply.

Schedule a new layout or edit a current layout using the Privacy components:

  • Communication Message Type corresponds to the privacy 1 component.
  • Consent Message Type corresponds to the privacy 2 component.
  • Terms & Conditions Message Type corresponds to the privacy 3 component.
  • General Message Type corresponds to the privacy 4 component.

User Privileges for Privacy Messages

To define the user rights for the ability to edit and view the online privacy messages add these privileges to the user.

  • online.privacy.message.edit
  • online.privacy.message.view
  1. In Site Manager go to Manage Users.
  2. Select Roles and assign the privileges to the role for the intended user.

Screen Labels

To access the Screen Labels go to the Content Tab > then select Screen labels from the left navigation. See Also: Screen Labels

Modify these screen labels to modify the header of the section.


Modify these screen labels to modify the link to the privacy message.

Note: Once you enter the text you MUST click outside the cell. And then select Apply and Save for the change to save. For the translations to take effect, you will need to clear all cache from your web browser before viewing the site.
