Add Open Table to myRestaurant Mobile

Add the ability to make a dining reservation through open table. If enabled and setup the Open Table icon displays next to the Restaurant on the search locations page. This feature is available in Mobile Sites and myRestaurant Mobile.

Frontend Search Locations page with Open Table Link

Steps to Add Open Table

Step 1. Upload the Open Table Icon to Manage Media

Step 2. Enable Open Table in Site Manager

Step 3. Add the Open Table ID to the Restaurant Attributes


Step 1. Add the Open Table Icon to the Manage Media Library

  1. In Site Manager go to the Content tab.
  2. Select Manage Media from the left navigation
  3. Upload the Open Table Icon and locate the image URL, copy and paste this URL for the next step.
    Note. If the image URL is not appearing, expand the browser window.

Step 2. Enable Open Table

  1. In Site Manager go to Site Settings.
  2. Select Manage Sites from the left navigation.
  3. Select the site's Front End Settings > and select the Room Service Settings tab
  4. Turn on OpenTable Access: On
  5. Open Table Link Icon: Copy and paste the URL from an image uploaded via Manage Media for the Open Table Link Icon

Step 3. Add Open Table ID in Manage Sites Elements

  1. In Site Manager go to the Content tab.
  2. Select the site from the Edit Site drop down.
  3. Select Manage Site Elements from the left navigation.
  4. Select Restaurant from the type drop down
  5. Select Find to list the Restaurants.
  6. Next to the Restaurant select the Edit option.

  7. Go to the Attributes tab and enter:
  • OpenTable Restaurant ID: An ID from OpenTable that relates the MCP/MC restaurant to OpenTable
  • OpenTable Restaurant Title: A title that will show as a link below the OpenTable Form generated

How it works:

  • When the Enable OpenTable Access is turned off, the Open Table Link Icon parameter will not show and all Open Table Links do not display.
  • When the Enable OpenTable Access is turned on but the OpenTable Restaurant ID is not specified for the restaurant, the link will not display.
