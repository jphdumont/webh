Change a Users Password

To change a user's password:

  1. Select the user's username from the list or select the "Edit" icon to the right of their name. A new page will appear with tabs for "Details" and "Associations".
  2. Select the "Details" tab.
  3. Enter the new password in the "WebHotel Administrator's Password" text field. The password must meet the following criteria:
  • It must be at least 8 characters long.
  • It must contain one alphabetic character [a-zA-Z]
  • It must contain one numeric character [0-9]
  • It must contain one special character from this set: ` ! @ $ % ^ & * ( ) - _ = + [ ] ; : ' " , < . > / ?
  • It must not contain spaces.
  • It must not begin with an exclamation point or a question mark.
  • The first three letters cannot be the same.
  • The new password cannot be the same as your last four passwords.
    1. Re-enter the password in the "Confirm WebHotel Administrator Password".
    2. Select "Save Attributes" to save the new password.
    3. Select "Back to Users List" to return to the previous page.
