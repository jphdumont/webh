Getting Around Site Manager

Accessing Site Manager URL

There are 2 factors to consider about Content the Admin URL and User Rights:

Admin Site URL: There are 2 ways to access the Site Manager Admin

  1. Login to Site Manager using the default site URL. This is the best way to access all the content for the site.
  2. Login using another URL and access the site from the drop down list and the user will only see content configured for the default site.

User Rights: What site is associated with your login credentials and who uploads the content with the Global Images folder affects what content the user can access.
