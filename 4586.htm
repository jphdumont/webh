Add the Order Information Bar to the Page Layout

  1. In Site Manager go to the Content tab and select the site from the Edit Site drown list.
  2. Select Sections & Pages from the left navigation.
  3. Select Edit next to the Online Ordering Pages, select Edit

Related Display Option Parameters

To access these parameters go to Site Settings in the site's Frontend Settings > Display Options:

Order Date/Time in another page.

Turn on to enable the Date and Time to display on another page. Turn off this parameter, the date and time would be selected in the same page as select store. Go to Sections and Pages Search Location and Results page and edit layout to add the component "Search Location Result List and Map" to the page. Show next available date and time.

Access this parameter settings in the site's frontend settings Site Manager > Site Settings > Manage Sites > Front end > Display Options > Confirm before Change Location

Confirm Before Change Location.

Turn on to enable the confirmation before Change Location.

Combined Order Summary and Checkout Page

Turn on to combine the order summary and checkout page.

When Combined Order Summary and Checkout Page is turned on, then the user is redirected to ReviewAndCheckout.jsp

Screen Label for Start Your Order Button

To change screen label for the Start your Order Button:

  1. In Site Manager go to the Content tab and select the site from the Edit Site drop down list.
  2. Select Screen Labels from the left navigation.
  3. Select the My Restaurant Web My Order group from the drop down list.
  4. Find the myres.order.info.btn.start change the text for the each specific language.
