Responsive Accordion Style Checkout Page

A responsive accordion style checkout page collapses the checkout steps into collapsible groups. This is responsive and works on mobile devices. The site must have a responsive template activated. Then edit the page layout for the reviewAndCheckout.jsp page to include the accordion checkout page.
See Also: Enable Online Ordering and Setup External Payment Gateways

Note. When >768 the groups display open and <768 the groups display collapsed as pictured below.

Configure the Accordion Style Checkout page

  1. In Site Manager go to the Content tab.
  2. Select the Site from the Edit site drop down list and select Sections & Pages from the left navigation.
  3. Select the Edit option next to the MyRestaurant Online Ordering Pages

3. Select Edit next to the Checkout.jsp page.

4. On the page layout area add the Accordion Checkout Details component to the reviewAndCheckout.jsp page.

5. Set the Page Layout status to Approved.
