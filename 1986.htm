Setup A Key Encoder with OPERA2Go

Using OPERA2Go the Agent selects which key encoder in the Hotel to create the room key and the number of keys to make while checking in a Reservation. To setup and activate this feature the Site Administrator needs to turn this feature on, define the maximum number of keys, and add the terminal IDs as setup in OPERA.

See Also: The "IFC Workstation Setup Tab" topic in the OPERA help for more information about setting up the workstations.

Setup the Key Encoder in OPERA2Go

Step 1. Enable the Key Encoder in OPERA2Go Site Settings

  1. To access this feature go to Site Manager > go to the Site Settings tab > select Manage Sites from the left navigation > choose the OPERA2Go site and select Front End Settings > and select the OPERA2Go Site Settings tab.
  2. Go to enable the Enable key Encoder Location feature, select the "On" radio button.

Step 2. Define the Maximum Number of Keys

This is the maximum number of keys the (OPERA2Go) user can request per Reservation.

  1. To access this feature go to Site Manager > go to the Site Settings tab > select Manage Sites from the left navigation > choose the OPERA2Go site, select Front End Settings > and select the OPERA2Go Site Settings tab.
  2. Define the No. of Keys with a maximum number of keys cut per Reservation.

Step 3. Add a Key Encoding Terminal to OPERA2Go

Access the settings for the Key Encoder Terminal from Site Settings > Manage Sites from the left navigation > choose the OPERA2Go site > select Front End Settings > select the OPERA2Go Site Settings > then select the tab Key Encoder Terminal.

Add a key encoding terminal to OPERA2Go.

Edit the key encoder Terminal ID, display label, and select if it is active or not.

Delete the key encoder terminal from OPERA2Go.

To Add a Terminal to OPERA2Go

  1. Select the symbol to add a key encoder terminal.

  2. Define the Terminal ID, the display label (name), and select "On" to make it active.
  3. Select "OK" and Save Settings.

Terminal ID. Terminal ID is the workstation ID in OPERA that will be passed to identify the correct encoder. See Also. For corresponding information in OPERA see The "IFC Workstation Setup Tab" topic in the OPERA help.

Display Label. The display (name) label is what the Reservation Agent will see when prompted in OPERA2Go to choose which key encoder to use when making a key for the Guest. For example: To use names (labels) like "West Elevator" would be a descriptive name to explain to the location to the Reservation Agent.

Active. Select "Yes" to activate the key encoder terminal or select "No" to deactivate the key encoder.

OK. Select "OK" then select save settings to save.

Note. The key encoders are originally setup in OPERA.
See Also: The "IFC Workstation Setup Tab" topic in the OPERA help for more information about setting up the workstations.

(Frontend) Check-In Process in OPERA2Go To Create a Key Using the Remote Key Encoder

OPERA2Go uses the following process flow for check-in:

  1. Find the Reservation to check-in. (Select either the Due in icon or the Reservation icon from the OPERA2Go homepage), and select the Reservation from the list of Reservations.
  2. Then select the Check-in button or Proceed with Check In button.
  3. If the "Enable key Encoder Location" feature is set to "On".
  4. Enter the Number of Keys.
  5. Select the Continue to proceed with the check-in process.
  6. Pick up the key from the specified key encoder.
