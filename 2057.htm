To Add or Remove an Accompaning Guest to a Reservation

To add or delete an additional Guest name to the Reservation select the Accompany Guest button on the Edit Reservation screen.

Step 1. Look up the Reservation from the Due In Main Menu option, at the bottom of the page is the Accompany Guests button.

Step 2. Add or Remove an Accompany Guest.

Detach. Select the detach button next to the Guest name to remove the name.

Lookup Profile. To search by first or last name enter the name and select Lookup Profile.

Attach. To add the name as the accompany Guest.
