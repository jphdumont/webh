Adding Content in a Popup (Lightbox)

To create content for the website using a popup in the rich text editor

Within the Rich Text Editor, you will have access to the Popup icon that inserts content in a popup. Schedule Content to use on a page using the Rich Text editor. In Site Manager go to the Content tab > select Sections & Pages from the left navigation. Schedule new content based on the "Rich Text" asset type.

To schedule content using a Popup

  1. Go to Site Manager > Content > Sections & Pages.
  2. From the Site Sections list, select the Edit icon next to the page to schedule the content.
  3. From the Pages list, select the Edit icon next to the page to edit the content.
  4. On the "Areas" grid, select the Edit icon for the page area in which the content will display.
  5. Select the Schedule New Content button.
  6. Select a Start Date and Start Time for the content to appear on this page using the buttons next to these fields.
  7. Select the Asset Type down arrow and choose Rich Text from the drop down list.
  8. Select the Format down arrow and choose Rich Text-Editor.
  9. In the Rich Text Editor, enter the content for the popup.
  10. On the Rich Text Editor toolbar, select the popup icon .
  11. In the properties dialog box, define the width, height, and time to close options.

    Default Width (px). Define the width of the popup in pixels.

    Default Height (px). Define the height of the popup in pixels.

    Time Delay to Close (in ms). Define the time delay to close in milliseconds.

    For Mobile Site. Check this box if it is a Mobile Site.

Select OK to save and close the dialog box.

Select the Save Rich Text Asset button to schedule the new content.

To approve the scheduled content

  1. Select Edit next to the area that has the New Content scheduled.
  2. Select the Status down arrow and choose Approved from the drop-down list box.
  3. Go to the frontend of the website to test the popup is applied.

Note. Remember to clear the site's cache and the browser's cache to see the changes. See Also: Cache Stats
