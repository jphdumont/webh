Site Map

To Access the Site Map Page on the Website Frontend

Go to Site Manager > Content > Sections & Pages > Select a Business Page if there is a Site Map page setup it displays in the Business pages list. Or if a site map is not listed, Create a Business page, and select Site Map as the page type.
See Also: Pages

The Site Map tab allows you to activate the page to be listed on the site map and configure site map settings for the page.

Page Configuration

Display in Site Map. Select to display the page on the site map. If this check box is clear, the page will not be displayed on the site map.

Change Frequency. Select the down arrow and choose a frequency for generating the sitemap.xml file for the page and submitting this file to search engines. The sitemap.xml file is generated dynamically by specifying the <Root URL>/sitemap.xml (e.g., yourhotel.webhotel.com/sitemap.xml).

Priority. Select the down arrow and choose a page priority for generating the sitemap.xml file and submitting this file to search engines. The sitemap.xml file is generated dynamically by specifying the <Root URL>/sitemap.xml (e.g., yourhotel.webhotel.com/sitemap.xml).

Save. Select to save configuration changes.
