Integrating the webHotel Internet Booking Widget onto a Facebook Page

Create a Facebook page with a webHotel Facebook Booking widget when the user enters the availability search criteria and selects Search the user goes directly to the booking path pages to book a room for the Hotel Chain or Property site.

Step 3. Edit the Facebook Template CSS (optional)

Step 4. Edit the Screen Labels (optional)

Step 5. lntegrate with the Hotel's Official Facebook Account (in Facebook)

Facebook Page with webHotel Booking Widget

Step 1. Configure the Facebook Booking widget (in Site Manager)

  1. In Site Manager go to Site Settings and select Manage Sites from the left navigation, select the site's Front End Settings go to the Facebook tab.
  2. Turn on the options to display on the Facebook webHotel Booking widget.
  3. Enter the Facebook Page URL. This page will route to the Facebook template page setup for the site under General Pages. The Facebook Page URL should have the following format: https://www.facebook.com/ljbtc?sk=app_167529213327720
    See Also: Facebook Page URLs
  4. Enter the Facebook Application ID.
  5. Promo Code. Define a default Facebook promo (rate) code automatically populates the Facebook booking widget.

    Note. There is the option to specify more then one Facebook page for the property or chain. Use the to define new webHotel Facebook page settings for the site.


Step 2. Define the page layout for the Facebook page (in Site Manager)

  1. In Site Manager, go to Content > Sections & Pages, select the General Pages Section > and select the Facebook Page. This pre-configured layout makes it easy to uncheck content areas that should not display or add content to areas to display around the booking widget.
  2. Access the Facebook page in the webHotel Site: http://webhotel-url/facebook/home.jsp

Preview the Page

Step 3. Edit the Facebook Page Template CSS

To edit the CSS for the Facebook template:

  1. In Site Manager go to the Motif Management select a site and select Templates from the left navigation.
  2. Select the Edit button next to the Facebook Template.

3. Edit the Facebook Template CSS

4. Save the Template

Step 4. Change the Screen Labels

  1. In Site Manager go to the Content tab and select Screen Labels from the left navigation.
  2. Select the site from the Edit Site drop down list.
  3. Search on the page for the labels beginning with search.fb. edit or define different languages for the Facebook Booking widget screen labels.

Step 5. lntegrate with the Hotel's Official Facebook Account

  1. Set up the Facebook app settings in http://developers.facebook.com/apps with the URL of the Facebook page.
    For example: http://webhotel-url/facebook/home.jsp
  2. Log on to your Facebook account and create a page with a link to your webHotel site.
  3. Execute the Facebook link to add the app to the current account.
