Privacy Options

The Privacy tab allows you to set the privacy mailing list option for the web site.

Display Mailing List. Select "On" to have the Mailing List check box selected by default. When a user selects "Join Today" from the landing page of the site, the "Join Our Newsletter" window will open with the Mailing List check box selected by default.
