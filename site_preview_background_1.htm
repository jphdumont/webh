Site Preview - Background Editor (webHotel)

Access the inline editor and go to the background panel by selecting the orange tab on the right side of the inline editor:

  1. Go to the Content tab and select a webHotel site from the Edit Site drop down list.
  2. Select Site Preview from the left navigation to access the inline editor.
  3. On the right side of the page select the Orange tab to access the background editor.
