Login Screen

Accessing OPERA2Go

To access the OPERA2Go application, on your tablet device, go to: http://www.o2go.yourcompany.microsdc.us

Log In to OPERA2Go

On the Login screen, you will enter your username and password and select a resort to log into. If an invalid username or password is entered, an error message will display on the screen to indicate that there was an invalid username/password entered, but the exact error (e.g., the username does not exist in OPERA or the password was invalid) will not be displayed for security reasons. For other login errors, the specific error will display.

Username. Enter your username.

Password. Enter your password.

Resort. Select the down arrow and choose a resort from the list. You will only see resorts displayed for which you have access.

Login. Select to log on to OPERA2Go.
