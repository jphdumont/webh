Change the Page Name in Site Manager

To change the name of a page in Site Manager:

  1. Go to the Content tab and select the Site from the Edit Site drop down list.
  2. Select Sections & Pages from the left navigation select edit next to the page.
  3. Then select the Details tab.
  4. And change the file name (Path).
