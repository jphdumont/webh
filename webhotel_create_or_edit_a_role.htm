Create or Edit a Role

Create a Role

To create a role

  1. In Site Manager, go to the Manage Users tab and select Roles from the list of options on the left.

  2. Under the Roles tab in the Name field, enter a name for the user role and select Create Role button. The new role will display in the grid below.

Edit a Role

To Edit a Role

  1. In Site Manager, go to the Manage Users tab and select Roles from the list of options on the left.

  2. Under the Roles tab, select the Edit button next to the role you wish to edit. The page will refresh and three tabs will display: Details, Users and Privileges. Details will be selected by default.

  3. Under the Details tab, you can edit the name and description of the role by entering text into these fields and selecting the Save Details button.

  4. Select the Users tab to assign specific users to the role by highlighting the user in the right pane and selecting the Add button. This assigns the user to the role and moves the user into the "Assigned Users" pane.

  5. Select the Privileges tab to assign privileges to the role by highlighting the privilege in the "Not Assigned" pane and selecting the Add button to move it into the "Assigned Privileges" pane.
