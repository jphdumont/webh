Turn on the Order Date and Time Selection for the Online Ordering Process

To enable the option for the user to enter the date and time during the first step of the ordering process in a myMenu Site.

Step 1. Turn on the parameter to enable

Step 2. Turn on the Component Configuration Option

Frontend Example :

Step 1. Turn on the parameter to enable

  1. In Site Manager go to Site Settings > Manage Sites and select the site's Frontend Settings.
  2. Then select the Menu Service Settings tab.
  3. Turn on the Enable Order Date / Time Selection parameter.

Step 2. Turn on the Component Configuration Option

To turn on the component configuration for the top navigation select the order date and time in Site Manager:

  1. Go to the Content Tab.
  2. Select the site from the Edit Site drop down list.
  3. Select Sections & Pages from the left navigation.
  4. Select Edit next to the My Menu Mobile Online Ordering Pages.

5. Select Component Configuration for the Online Ordering Page.
