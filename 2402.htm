Facebook Page URLs

To link a Facebook Application ID to this Site's Facebook Booking Widget enter the Facebook information in the Frontend Settings > Facebook for the site. To design the Facebook page content including the Booking Widget, see also: Integrating the webHotel Booking Widget onto a Facebook Page

Note. This only applies to webHotel Sites.

To access in Site Manager go to Manage Sites and select the Frontend Settings for the Site. On the Facebook tab enter the Facebook Page URL and the Facebook Application ID.
Select the plus sign to add more Facebook Page URLs.
