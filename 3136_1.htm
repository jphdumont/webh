Integrate Campaign Mailings with Exact Target

Use Site Manager to create mailing lists based on Guest stay information (i.e based on Guest's with a specific rate code, arrival day, length of stay, etc.) then synchronize this mailing list with your Exact Target account to send the emails to the Guests.

Step 1. Create the Email Template

  1. In Site Manager go to the Content tab and select Campaign Email Template from the left navigation.
  2. Select Create Campaign Email Template.

Step 2. Define Exact Target as the Email service provider

  1. Select Exact Target from the service provider drop down list.
  2. Add any dynamic variables and the data will be imported into Exact Target.
  3. Save.

Step 3. Configure the Filters and select the template setup for the Exact Target Integration

  1. In Site Manager select the Content tab, then select Manage Campaign Configuration from the left navigation.
  2. Select the Campaign Mailing Template setup in the previous step.
  3. Add any filters to refine the list of recipients. For example, define arrival dates, select a specific reservation status, select a specific rate code, etc. to compile a list with only with the recipients that match this criteria. This list of recipients will sync into the exact target account.

Step 4. Add the Exact Target User Credentials

  1. In Site Manager go to the System Tools tab and select Web Service Settings from the left navigation.
  2. Select Add Service to define the new Exact Target Service.
  3. Select Exact Target WS (Web Service) from the Service Type drop down list and the host URL automatically populates host URL under the Exact target API call heading.
  4. Copy this URL to the Service Host field. (In this example the Service Host is webservice.s4.exacttarget.com.)
  5. Enter the User Information (credentials) from Exact Target.
  6. The other fields are optional.

Step 5. Select Exact Target in the Backend settings

  1. In Site Manager go to Site Settings > Manage Sites and select the site's Backend Settings
  2. On the Details tab select the ExactTarget Service Context from the drop down list. (This is populated by the web service setup in the previous Step 4.)
