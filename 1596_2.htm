Create a Restaurant and Create a Restaurant Menu (for webHotel)

These instructions are to create a menu with a stactic menu id, for webHotel. See Also: Enable Online Restaurant Ordering for a Mobile Site, Setup Dining Reservations for a Mobile Site, or Setup a myRestaurant Website.

To manually create a Restaurant for a Site:

  1. Go to Content > manage Site Elements > under Find a Site Element select a Restaurant from the Type Drop down menu.
  2. Then select Find to Edit an existing Restaurant, or select Create a new to add a Restaurant.
  3. Select Images to upload images.
  4. Select the Attributes tab to enter a description, external code (This is the store or restaurant id that is used in mapping this restaurant to its corresponding restaurant in the POS system.), currency, hours, address information.
  5. Select Save.

To Import Restaurant data from OPERA

The Hotel's Restaurants are setup in System Tools > Hospitality Products > Choose Type Restaurant.

Working with Menus for a Restaurant:

To find an existing Menu:

  1. Go to Content > manage Site Elements > under Find a Site Element select Restaurant Menu from the Type Drop down menu.
  2. Select a Restaurant and select Find to display the menus. if no menus display after you select find, then a menu is not setup.

To Edit a Menu:

  1. Find a menu, then from this list select a menu to edit.
  2. Select the Attributes tab to create the Restaurant Menu in a WYSIWYG editor. Add descriptions, images, and other formatting to create the menu.
  3. Select the Images tab to add an image.
    See Also: Element Images.

To create a new Menu:

  1. Go to Content > manage Site Elements > under Find a Site Element select Restaurant Menu from the Type Drop down menu. Select a Restaurant, then select Create New.
  2. Enter the Name and description of the Menu, select Type > Restaurant Menu select the Restaurant and select Save.
  3. Select the Attributes tab to create the Restaurant Menu in a WYSIWYG editor. Add descriptions, images, and other formatting to create the menu.
  4. Select the Images tab to add an image.

To Display on the Frontend of the Website

Use the web services to access the menu id.
