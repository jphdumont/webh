Post a Charge

To post a charge to a folio

  1. From the Dashboard screen, tap the Due Out, In House or the Reservation icon.

  2. From the Search screen, enter a last name or a confirmation number and tap the Search button.

  3. On the Lookup Results screen, tap the View Folio button.

  4. From View Folio screen, tap the Post Charges button.

  5. From the Post Transaction screen, select a transaction code and/or an article and enter an amount for the charge. Tap the Post Charge button to post the charge.

Supplement Text. Enter information about the posted transaction in Supplement Text box. It is mandatory to enter notes here, if there is a negative charge posted.
