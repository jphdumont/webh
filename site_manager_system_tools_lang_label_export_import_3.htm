Language Label Export/Import

To access this feature in Site Manager go the Content tab and then select Language Label Export / Import from the left navigation.

Use this page to export language labels to a spreadsheet or import a language label spreadsheet for your site.

Choose site: Select the down arrow and choose a site from the drop-down list.

Choose From Language: Select the down arrow and choose a "From" language from the drop-down list. This is the language to translate the labels from.

Choose To Language: Select the down arrow and choose a "To" language from the drop-down list. This is the language to translate the labels to.

Choose action

Under this section, you can choose to either export a language label to a spreadsheet or import a language label spreadsheet to a site.

File Name: Select the Browse button and choose a spreadsheet to import to the site. Select the Import button to import the spreadsheet.

Button Functions

Export. Select to export language labels to a spreadsheet.

Import. Select to import a language label spreadsheet to a site. The file type must have an "xls" extension.

Exporting and Importing Language Labels

To export language labels to a spreadsheet

  1. From the "Choose site" field, select a site.
  2. From the "Choose From Language" field, select a language to translate the labels from.
  3. From the "Choose To Language" field, select a language to translate the labels to.
  4. Select the Export button.
  5. Open the exported spreadsheet and edit it by changing the values under the "Label to Value" column to existing translations for the language.
  6. Save the spreadsheet file.

To import a language label spreadsheet

  1. From the "Choose site" field, select a site.
  2. Select the Browse button and choose an Excel file to import.

    Note: The file type must have an "xls" extension.

  3. Select the Import button.
