Configure the Upgrade Option for a Mobile Site

The Upgrade option is available to a Guest who participates in Membership programs. This feature provides the ability to display upgrades available to members configured in OPERA on your Hotel's mobile website for the Guest to be able to upgrade their Reservation.

To configure the room upgrade feature for your mobile site, follow the steps below. The steps are divided into three parts.

Step 1: Configure the Upsell Rule in OPERA

To configure Upsell Rules in OPERA

  1. In OPERA PMS, set the Front Desk > Functions > Reservation Upsell application function to Y. Also, ensure that your user has the RESERVATION UPSELL CONFIGURATION permission granted.
  2. Go to Setup > Configuration > Rate Management > Upsell to open the Upsell Configuration screen.
  3. From the Upsell Configuration screen, select your property from the Property field and select the Rules button.
  4. On the Upsell Rules screen, select the New button.
  5. On the Upsell Rules screen, complete the following required fields:

    Rule Code (Enter a code to identify the rule.)

    Begin Date (Select a begin date for the rule using the calendar button.)

    From Room Type (Select the down arrow and choose a "from" room type from the list of values. This indicates the original room type requested by the customer.)

    To Room Type (Select the down arrow and choose a "to" room type from the list of values. This indicates the upsell room category.)

    Formula (Select the down arrow and choose a formula to apply to the rule. The formula selected here will be used to calculate the upsell amount that is offered. The following formulas are available: Flat amount added to original rate; % of difference added to original rate; and % of original rate added to original rate.)

    Upsell Amount (Enter an upsell amount to be added to the Base Type's rate amount.)

    Currency Code (Select the down arrow and choose a currency code to be used for the upsell amount.)

  6. Select OK to save the rule. You can also select the Test button to test the upsell rule with a rate code and see the results of the rule.

Step 2: Enable the Room Upsell and Room Upgrade Features in Site Manager for Mobile

To configure in Site Manager:

  1. Go to Site Settings > Manage Sites and select the Front End for your mobile site.
  2. Select the Site Settings > Manage Sites > and select the Front End > Search Rooms tab and select the Room Upsell "On" radio button.
  3. Next, go to the Membership tab and select the Room Upgrade "On" radio button. This will enable the "Upsell" link on the Reservation confirmation page of your mobile site.

Step 3: Test the Site

To test the Upsell / Upgrade feature on your site:

Tip. Make sure you make a Reservation associated with a membership that has upgrade options configured in OPERA.

  1. Go to your mobile site and search for a reservation. "You are eligible for Upgrade, would you like to review the Room Upgrade Options?"
  2. Select this option to go to the available upgrades, then the Reservation screen will be updated with the new upsell options.
  3. Click the Update Now button to update your reservation.
  4. If the Upsell or Room Upgrade is successful, a confirmation page will display with a message indicating the reservation has been updated.
