Scheduling a PrettyPhotoLightBox

PrettyPhoto LightBox Galleries are setup for a page under Content > Sections & Pages > select General Pages, then select PhotoTour and edit. To schedule new content based on the PrettyPhotoLightbox asset type, select the PrettyPhotoLightbox format from the drop down list. The PrettyPhotoLightbox format supports images.

To schedule a PrettyPhotoLightbox Content

  1. Go to Site Manager > Content > Sections & Pages > select General Pages > select photo gallery.
  2. From the Site Sections grid, select the Edit icon for the page section to schedule the content in.
  3. On the Pages grid, select the Edit icon for the page that the content will be scheduled in.
  4. On the Areas grid, select the Edit icon for the page area where the PrettyPhotoLightBox will display.
  5. Select the Schedule New Content button.
  6. Select a Start Date and Start Time for the PrettyPhotoLightbox to appear on this page.
  7. Select the Asset Type down arrow and choose PrettyPhotoLightbox from the drop-down list.
  8. Under Asset Options, select the Add Image button to start the Image Browser and double click an image file to add it to the photo gallery. Repeat this step for each image in the gallery.
  9. Refer to the "To Setup the PrettyPhoto LightBox Options" section below for a detailed description on how to configure the options for your look and feel.
  10. When you are finished adding images and configuring properties for the PrettyPhotoLightBox, select the Save Pretty Photo button.

Tip: After adding images, you can reorder the sequence of the images, delete an image, edit the captions for an image (refer to the "To Setup the PrettyPhoto LightBox Gallery" heading below). Then select the Save Pretty Photo Gallery button to save changes to the images.

To Approve and Preview the Scheduled Content

  1. Under the Areas grid, select the area where the gallery is scheduled.
  2. Under Live Content, select the Edit button.
  3. Select the Status down arrow and choose Approved from the drop-down list box.
  4. Select the Preview in Page button to preview the gallery.

Setup the Pretty Photo LightBox Options

  1. Go to Content > Section & Pages > General Pages > then select the Photo Tour page.
  2. Select the Edit Content Details button to access the properties to configure for the PrettyPhotoLightBox photo gallery:

Add Image. Select to add images to the PrettyPhotoBox, upload from your computer or select an image from the media library.

Animation Speed. Select the fast, normal, or slow option.

Slide Show Time. Enter the animation speed in milliseconds, the default is 5000.

Auto Play. Select True or False.

Opacity (0-1). Enter a value to set how transparent the overlay of the lightbox displays.

Theme. Choose from several options:


Default Theme


Facebook Theme


Light Rounded Theme


Light Square Theme


Dark Rounded


Dark Square

Width. Maximum width of the image when "Expand this image" is selected on the top right of the lightbox image.

Unit Width. Define the width of the image as displays on the page before it is selected and enlarges as a lightbox.

Unit Height. Define the height of the image as displays on the page before it is selected and enlarges as a lightbox.

Order. Enter a numeric value for the order of the image.

Image Name. File Name of the image.

Element Type. Select the Photo 1 or Photo 2 option.

Title. Enter a title for your image which appears under the image in the PrettyPhotoLightBox

Description. Define a description, displays under the image in the PrettyPhotoLightBox. Enter up to 140 characters.

Edit Labels. Specify an image label.

Delete. Select the Delete button to remove an image.
