OPERA2Go Application Settings

Go to Manage Sites > Front End > Select your OPERA2Go site > then go to the Application Settings tab. This value determines the amount of session time when the users log into OPERA2Go before they we be asked to log in again due to inactivity.

Agent Allowed Session Inactivity Period (in minutes): Enter the amount of minutes for each session. When the session is idle for this amount of minutes any links will redirect you to login "Session Timed Out, Please log in again."

Save Settings. Select to save any changes.
