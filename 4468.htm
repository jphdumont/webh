Site Elements Types

To show or hide the Site Elements Types in the Content > Manage Sites Elements Type drop down list:

Step 1. Turn on or off from Site Element Types in the Backend Settings

  1. In Site Manager go to the Site Settings.
  2. Select Manage Sites from the left navigation.
  3. Select the Site's Backend Settings.

4. Select the Site Element Types tab to turn on and off the Site Element Types.

5. Select Save Settings to save the changes.

Step 2. Go to the Content > Manage Site Elements

  1. In Site Manager go to the Content tab.
  2. Select Manage Site Elements to see the Site Element Types that are turned on display in the Type drop down list.
