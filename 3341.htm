Add Home Store Location to My Account Page

To add the Home Store Location componet to the My Account page:

  • Step 1. Turn on the parameter to show the home store location feature.
  • Step 2. Add the Home Store Location component to the My Account (my profile) page

Step 1. Turn on the parameter to show the home store location feature

In Site Manager go to the Site's Frontend Settings and select the Display Options tab to turn on the Show Home Store Location.

Step 2. To add the Home Store Location component to the My Account (my profile) page

  1. In Site Manager go to the Content tab, select the Site to edit from the drop down list.
  2. Select Sections & Pages from the left navigation.
  3. Select Edit for the My Restaurant Web Account Pages and select Edit next to the My Account page (myres/profile/myAccount.jsp).
  4. Select Schedule a New Layout to make a new layout for the My Account page.
  5. To create a page layout then select Edit and Edit Layout.
  6. Create a page layout to add the Home Store Component to the page layout.
