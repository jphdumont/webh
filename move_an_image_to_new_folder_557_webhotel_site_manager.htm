Move an Image to a New Folder

To move an image to a new folder:

  1. Navigate to the folder the image is in.
  2. Locate the image to be removed.
  3. Click and hold the left mouse button over the image.
  4. Drag the image to the navigation portion of the image browser.
  5. Release the mouse button when the image is over the folder it should be moved to. The image will be moved to the new folder.
