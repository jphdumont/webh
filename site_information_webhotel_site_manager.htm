Site Information

The Site Information tab allows you to change the name of a property or chain site.

To access the Site Information, in Site Manager go to the Manage Sites list and click on the site name. Then select the Site Information tab.

Property Name. Enter the property name.

Save. Select to apply the name change to the property or chain site.
