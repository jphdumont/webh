How to Configure Payment Types for myMenu

How to setup payment method IDs for myMenu:

Step 1. The values for the Channel Method payment IDs are setup in myCentral.

Step 2. Configure in Site Manager

Step 3. View on the myMenu Frontend

Step 4. The payment is processed at the MICROS POS.

Configure in myCentral

To set the values in myCentral go to Sales Channels > Web > Payment, select FulfillmentTimeType > then define PaymentMethodId values in the Payment Methods Display text field. Each value is unique and sent in the confirmation email.

Configure in Site Manager

In Site Manager go to the myMenu site Frontend Settings and select the Menu Service Settings tab. Add the Channel Payment Method IDs from myCentral to the respective Channel Payment Method ID fields in Site Manager.

Channel Payment Method ID. Channel Payment Method ID for Cash A Channel Payment Method ID to be passed to the Web Service Request when placing order using Cash as payment.

Channel Payment Method ID for Bill To Room. Channel Payment Method ID for Cash is a Channel Payment Method ID passed to the Web Service Request when the user places an order using Bill to Room as payment.

Channel Payment Method ID for Credit Card. Channel Payment Method ID for Credit Card is a Channel Payment Method ID passed to the Web Service Request when the user places an order using Credit Card as payment.

Frontend Example

Pay Here. Select Pay here to use cash, check or give your credit card to the Server to process at the MICROS POS.

Bill to Room. Select to bill the meal to the hotel room transaction.

Pay by Card. Select to pay for the meal by credit card.
