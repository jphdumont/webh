Add Images to Packages in Mobile Sites

In Site Manager to add images to packages:

  1. Go to the Content tab and select the Site from the Edit Site drop down list.
  2. Select Manage Site Elements from the left navigation.
  3. Select Packages from the Type drop down list.
  4. And select Find.

5. Select Edit next to the Package to add images.



6. Add up to 10 Images.
