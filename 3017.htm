Restrict the Pickup of Alcoholic Beverages

myRestaurant Sites provide the ability to display all restaurant menu items with the ability to restrict specific category IDs from Take Out orders (pickup). If configured, this provides the ability to view the alcoholic beverages for sale at the Restaurant, but does not allow the user to order for Take Out. If the user selects

To prevent the online ordering of alcoholic beverages by restricting specific Category IDs from pickup:

  1. In Site Manager select the Frontend Settings for the myRestaurant Site.
  2. Go to the Online Ordering Settings and enter the Restricted Category IDs
  3. Input the Restricted Category IDs: 513 (For example: Alcoholic Beverages)
    Specify the Category Ids for items on the menu that should be restricted for Take Out (pickup). If the user selects pre-order dine in and places these items in the cart then switches to take out when placing the order the items will be validated to confirm the items in the cart are available for take out.
    See Also: Online Ordering Settings or Menu Service Settings (for myMenu Sites)

Note: This is an example and will be different based on the site's configuration.

Restricted Items on the Website Frontend

  1. Navigate to the myRestaurant website.
  2. Select Order Take Out > Select Location > Select Pickup Date and Time > go to Menu Page > select Restricted Category (for example: alcoholic beverages)
  3. Popup Displays "This item is restricted to take out"

Configure in Screen Labels related to Restricted Items (for pickup)

To customize the message for restricted items not available for pickup:

  1. In Site Manager, go to the Content tab.
  2. Select the Site from the Edit Site drop down list on the top left of the Site Manager.
  3. Select Screen Labels from the left navigation.
  4. Select my Restaurant Web my Order
  5. Navigate to the screen labels and Edit in the desired language.
