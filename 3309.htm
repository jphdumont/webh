Configure Group Ordering for myRestaurant Sites

To place an order for a Group of people use the group ordering option on the myRestaurant Sites. This enables the website user to create a group orders easily on the frontend of the website. The coordinator (for the group) adds email addresses to invite the others in the group to place the order on the website. An invitation email is sent to the invite the Guest to place the order on the website. With the ability to add multiple email addresses and send email confirmations with a link to the online order status. Group Ordering Links are automatically added to the navigation header of the Site. Group Ordering is always active and the related page components can be added to custom page layouts on the site.

See Also: Group Ordering Frontend Example Screen Flow

Options to customize Group Ordering:

  1. Custom Page Layouts: Create custom page layouts with the group ordering page components.
  2. Navigation: Define which Site Navigation to use: simple header, header, or the navigation links component (each contains the group ordering link).
  3. Parameters: Frontend Settings > Online Ordering define Group Order Parameters define Time Range and Order Limits.
  4. Screen Labels: Use Screen labels to custom fields on the frontend of the site.

Step 1. Edit the Layout of the Group Ordering Pages

  1. In Site Manager, go to the Content tab select the site to edit from the Edit Site drop down list.
  2. Select Sections & Pages from the left navigation.

  3. Select the myRestaurant Web Group Ordering Pages, select Edit next to the page to make changes to page layout.
  4. Select Schedule New Layout.
  5. Define the date and select Create.
  6. Select Edit Layout and define the page components on the page layout.

Step 1. Add Navigation for Group Ordering Pages

Navigation for Group Ordering is built into the simple header, header, and navigation links components. Use any of these page components to add Group Ordering option to the top navigation of the site.

Step 2. Customize the Group Ordering Pages

Use the Group Ordering page components to create custom page layouts for the Group Ordering pages.

Step 3. Define the Parameters for Group Ordering

  1. In Site Manager go to Site Settings and select Manage Sites from the left navigation.
  2. Select the Site's Frontend Settings and go to the Online Ordering tab.
  3. Define the values for the Group Ordering - Time Range and the Group Ordering - Order Limit

Step 4. Customize the Text Using Screen Labels

  1. In Site Manager go to the Content tab and select Screen Labels from the left navigation
  2. Then select the Group My Restaurant Web Group Ordering from the drop down list.
  3. Edit the text for the respective text field on the frontend of the site.

    Note. To save the changes you MUST click outside the cell, then select Apply and then Save.

    See Also: Screen Labels

To Customize the Text for the Error Message must enter for required fields use the Screen Label

In addition to the specific screen labels for the group ordering pages, update the error message when the field is required and not entered by the user.
