Language Settings

To access the Language Settings go to System Tools > select Language Settings from the left navigation.

To edit a language's details:

  1. Locate in the list of languages the language whose details you wish to modify.
  2. Select the "Edit Details" icon. The page will refresh and you will see a list of the details of the language.
  3. To change the name of the language, enter a new name in the "Name" text field.
  4. To change the ISO/2 code of the language, enter a new code in the "Code (ISO/2)" text field.
  5. To change the ISO/3 code of the language, enter a new code in the "Code (ISO/3)" text field.
  6. To change the ISO/5 code of the language, enter a new code in the "Code (ISO/5)" text field.
  7. To change the OWS code of the language, enter a new code in the "Code (OWS)" text field.
  8. To make the language active, select the "Active" checkbox.
  9. To make the language inactive, deselect the "Active" checkbox.
  10. To set the language as the default language, select the Default checkbox.
  11. Select Save Attributes to save your changes.

To Add Languages to the Available Languages for the website:

Go to Manage Sites > select the site's Backend Settings > and go to the Language Settings tab.
