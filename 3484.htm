Display Special Instructions on Room Service Menu Items

Enable the ability for the Guest to add special instructions to menu items when ordering room service on a Mobile site. The instructions can be added on the order or item level in myCentral.

Frontend Example

To Enable in Site Manager

  1. In Site Manager go to Manage Sites and select the Site's Frontend Settings.
  2. Select the Display Options and turn on the "Display Special Instructions" setting to On.
  3. Save.
