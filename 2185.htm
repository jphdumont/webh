Create a myRestaurant Website

There are several ways to work with Restaurants using MCP:

  1. One way MCP works with Restaurants is to enable online ordering for a webHotel or Mobile Site. In Site Manager you can create a Hospitality Import from OPERA (or myCentral) and create menus and restaurants that can be accessed in the webHotel or Mobile website. See Topic: Enable Online Ordering
  2. Room Service is another way MCP works with Restaurants, to enable Room Service Ordering for a Mobile Site
    See Topic: Setup Dining Reservations for a Mobile Site
  3. To create a complete Restaurant Website or Restaurant Mobile Website follow these steps:

Step 1. Create a myRestaurant Website

  1. In Site Manager go to Site Settings and select Create a Site:
  2. Select the Web Service Context.
  3. Optional Step. Enter an additional user name if you would like an additional email sent when the site is created.
  4. For Site for Application select My Restaurant Web
  5. Select a Room Service Context for the site. The room service context is the link to the myCentral and pulls the products from the myCentral database. (This option appears after selecting "My Restaurant Web".)
  6. Select any Channel Code and a Property.
  7. Select myRestaurantWeb Cloner under Use Site Settings from: myRestaurantWeb Cloner.
  8. Enter a Site Name and a Site domain.
  9. Define a Site Subdomain.
  10. Select a Template.
  11. Select the Submit button to create the site.
  12. Activate the Site by selecting the Active Checkbox.

Step 2. Activate the Room Service Context

  1. In Site Manager > go to Manage Sites and select the Site Settings > Back-end Settings for the site.
  2. On the Details tab, define the Room Service Context. See Also: Backend Configuration Details


Step 3. Configure the Display Options for the Restaurant Site

In Site Manager > go to Manage Sites and select the Site Settings > Front-end Settings for the site > select the Display Options tab.

Display Logo on Header. Select on or off to display the logo on the header.

Header Logo Width. Define the header logo width.

Header Logo Height. Define the header logo height.

Note. If zero is entered for both the width and height the image will appear at the original size.

Display Site Background Image. Select on or off to display the site background image.

Image Server Transparent Parameter. Select "On" for Image server transparent or "Off" to turn off the Image Server transparency.

Site Background Image Width. Define the number of pixels for the background image width.

Site Background Image Height. Define the number of pixels for the background image height.

Step 4. Define the Navigation Links

  1. In Site Manager go to Manage Sites and select the Site Settings > Frontend Settings for the myRestaurant site.
  2. On Navigation Links Settings, turn on and off links to display or hide.


Working with Navigation Links

  • There are (static) business pages or dynamic pages.
  • To add a business (static) page link select the icon, then choose an internal page to link to or define an external URL.
  • To move the location of the navigation link, select theicon and drag and drop the link to re-arrange the order.

    Dynamic Links: Dynamic catagory links are links that pull information dynamically from myCentral.

  • Locations
  • Menu
  • Catering
  • My Order

Step 5. Import the Restaurant Information

  1. In Site Manager > under System Tools > select Hospitality Products from the left navigation.
  2. On the Hospitality Products page: Select a Site, a Property and select the type Restaurants to import the Restaurants to Site Manager.
    See Also: Hospitality Products

Step 6. Edit the Restaurant Information in Site Manager

  1. Site Manager > Content > Manage Site Elements select Restaurants from the type drop down list and select Find.

    Drag and Drop the order of the Restaurants to change the order on the frontend of the website.

  1. Select Edit to:
  • Add an image
  • Restaurant Attributes. Define Attributes per property.
    To enable online ordering enter Y in the online ordering text field.
    See Also: Enable Online Restaurant Ordering and Setup Dining Reservations
  • Define Language Specific Attributes for the Restaurant define Alternate text, Short description, Contact us, Address line, Restaurant Hours, and Long Description.
  • Select Save to save the changes.

Step 7. Modify the Template Colors

  1. In Site Manager go to Site Settings and select Create a Site:
  2. In Site Manager > go to Motif Management > select the Template Colors
  3. Choose an item to customize and set the desired color.
  4. Save the selected color.

Step 8. Modify the Fonts

  1. In Site Manager > go to Motif Management > select the Template Fonts
  2. On Font Type tab, select the desired Font Family and Save.
  3. On Font Size tab, select the desired size and Save.
    See Also: Template Fonts
