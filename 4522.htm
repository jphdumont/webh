Enable the My Addresses and Stores from the My Account Page on the Frontend (myRestaurant Sites)

Turn on or off the ability to access the My Addresses and Stores page from the My Account page on the frontend of the site.

See Also Account Related Topics:

  • Working with the Account login Page for myRestaurant Sites

Step 1. Enable the Addresses & Stores on My Account Pages

  1. In Site Manager go to Site Settings tab.
  2. Select Manage Sites from the left navigation.
  3. Select the Site's Frontend Settings and go to the Display Options tab.
  4. Turn on the Display Addresses & Stores on My Account Pages parameter.

To Access the My Addresses and Stores Page in Site Manager:

  1. Go to the Content tab.
  2. Select the site from the Edit site drop down menu.
  3. Select Edit next to the web Ordering Sections & Pages.
  4. Edit the My Addresses & Stores
