Mobile Page Types and Ways to Customize Mobile Sites

MICROS Mobile (myStayManager) Sites are installed with pre-defined page types:

  • Booking Path Pages: Frontend | Backend
  • Account Pages: Frontend | Backend
  • Business Pages: Frontend | Backend
  • Marketing Pages: Frontend | Backend

To customize mobile sites using Site Manager:

  • Add Templates in Site Manager
  • Component Configuration
  • Customize the CSS
  • Create Custom Page Layouts
  • Translate and change the text using Screen Labels
