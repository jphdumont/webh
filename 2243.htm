Site Meta Tags

To access this feature, in Site Manager go to Manage Sites select the site and go to Backend Settings, select the Site Meta Tags tab. Use standard markup syntax to add the meta tags for the entire site.

To add meta tags to individual pages in Site Manager go to the Content tab, select Sections & Pages find the page select Edit and go to the SEO tab. See Also: Pages SEO
