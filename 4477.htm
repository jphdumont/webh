Display Nutritional Information on Menu Items

To display nutritional information on the product pages on the myRestaurant sites:

Step 1. Configure myCentral by defining the key value pair to include "ENERGY" as the key attribute to define the value as the nutritional information to display.

Step 2. Turn ON the Frontend Settings > Display Options > Enable Nutritional Information parameter

Step 3. The nutritional information displays when using any of these components:

  • Menu Grid
  • Menu Panel
  • Upsell Items component (Upsell items when displaying item details by modal screen.)

Step 1. myCentral Configuration (Required)

Configure myCentral by defining the key value paris to include "ENERGY" as the key attribute to define the value as the nutritional information to display.

On the product page in myCentral define the key / value pair for the nutritional information.

Step 2. Turn ON the Frontend Settings > Display Options > Enable Nutritional Information parameter

  1. In Site Manager go to Site Settings tab.
  2. Select Manage Sites from the left navigation.
  3. Select the Site's Frontend Settings > Display Options
  4. Enable Nutritional Information by turning on the parameter.

Step 3. Nutritional Information Page Layout Components

  1. In Site Manager go to the Content tab.
  2. Select the site from the edit site drop down list.
  3. Select Sections & Pages from the left navigation.
  4. Edit the page layout to include
  • Menu Grid
  • Menu Panel
  • Upsell Items component
