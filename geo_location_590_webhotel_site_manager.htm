Geo Location

Enter the city and country or countries the user must, or must not be, located in.

Select the country from the "Country" drop-down list box. If there are specific cities to be used in user group, valid cities will appear in the "City" list field after you type two characters in that field. To include all locations in the specified country/city combination, select "Include any". To exclude all locations in the specified country/city combination, select "Exclude all". Select "Add" to add the combination to the user group.

To remove a country/city combination, select the "Remove" icon to the right of the combination in the list.

Add a timeframe constraint. This could be the same as the experience timeframe, a start and end date, in the last number of specified days, or more than a specified number of days ago.
