Configure Pre-Order Buttons for Dine In

To setup the option for a Guest to pre-order for dining in the Restaurant. Active the pre-order button and configure if the dine-in time displays.

To turn on the pre-order buttons:

  1. In Site Manager go to the Content tab and select the Site to edit from the drop down menu on the top left of the page.
  2. Go to Sections & Pages, select the General Pages section.
  3. Find the Homepage and select edit.
  4. Select to Edit the Template to add the (Page Area) components to the page. The components are the Pre-order for Dine-In Button or the Order for Take Out Button.

Turn on or off the Setting to Display the Dine-in Times

For dine-in orders there is a parameter setting to hide or show the dine-in time on the frontend of the site. To access this parameter:

  1. In Site Manager go to the Site's Frontend Settings > Display Options > Display Time for Dine In. By default the last time slot available for the dine-in date displays (this value is returned by myCentral).
