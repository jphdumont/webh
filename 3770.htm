Configure Responsive Menu Grid for the Menu Page

The menu grid is a responsive menu component. When the screen size is less than 768 pixels then the menu grid automatically becomes a menu list suitable for viewing on a mobile device.

To apply a responsive the responsive menu grid:

Step 1. In Motif Management Activate the Responsive Template for the Site.

Step 2. Add the Menu Grid Component to the Page Layout.

Step 3. View Frontend Example

Step 1. In Motif Management Activate the Responsive Template for the Site.

  1. In Site Manager go to the Motif Management tab and select Templates from the left navigation.
  2. Select the radio button next to the Responsive Template and select the Activate button.

Step 2. Add the Menu Grid and the Accordion Menu List to the Page Layout

View Site Manager Navigation to Online Ordering Pages

  1. In Site Manager, go to the Content tab. Select a Site to edit.
  2. From the left navigation select Sections & Pages, select Edit next to the My Restaurant Web Online Ordering Pages.

3. Select Edit next to the Menu Page (myres/menu/menu.jsp) page from the list.

4. Drag the Accordion Menu and Menu Grid on to the page layout

5. Select Apply and Save Layout to apply the changes to the layout.
