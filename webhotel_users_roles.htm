Roles

The Roles tab displays the assigned roles for a Site Manager user.

Assigned Roles: This list box contains the list of roles to which the user is assigned.

Add: Select this button to move a role from the "Not Assigned" list box to the "Assigned Roles" list box.

Remove: Select this button to move a roles from the "Assigned Roles" list box to the "Not Assigned" list box.

Not Assigned: This list box contains the list of roles to which the user is currently assigned.
