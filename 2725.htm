Troubleshooting Potential Errors During an Upgrade

Clearing cache is always a good starting point. Follow these 3 steps to clear the cache:

  • Clear Site Cache
  • Clear Browser Cache
  • Close the Browser just to make sure the session closed and reopen a new browser window.

Java Heap Error

If a Java Heap Error occurs, this means the machine does not have enough physical memory to process the java code. Change the Heap

The default values :

  • Java permanent generation memory = 1024
  • Java Minimum Heap Size = 1024
  • Java Maximum Heap Size = 2048 (at Data Center if there is 8GB or more memory you can incease this to 4096)
