Define Required Fields for the Guest Details and Guest Profile Pages

Define which fields are required on the frontend of the Mobile site on the Guest Profile page during the booking process. The information the Guest updates on the profile screens on the MCP Mobile site updates the Guest profile in OPERA.


To customize the required or not required fields use these CSS classes in the stylesheet of the template:

  • input_field_required
  • input_field_optional

Guest Profile Page Frontend

Step 1. Define the fields as Required

  1. In Site Manager go to Manage Sites and select the site's Frontend Settings
  2. Select the Screen Fields tab and check the require / mandatory checkbox for each field

Step 2. Customize using CSS

To customize using CSS add these classes to the stylesheet:

  • input_field_required
  • input_field_optional
  1. In Site Manager go to the Motif Management
  2. Edit the Template and select the stylesheet
  3. Add the classes input_field_required and input_field_optional and define the style in CSS.
