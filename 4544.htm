Deal Promotion Registration Component

To add the ability for the user to Signup for Deals on the frontend of the site add the Deal Promotion Registration component to a custom page layout.

Required. Configuration is required in myCentral. It is required to define the Capillary web service context.


Frontend Example of Deals Signup

Step 1. Add Deal Promotion Registration Component to the Page Layout

  1. In Site Manager go to the Content tab and select the Site from the Edit Site drop down list.
  2. Select Sections & Pages from the left navigation.
  3. Add the Deal Promotion Registration component to any custom page layout.

Step 2. Select the Capillary Service Context in Backend Settings

  1. In Site Manager, go to Site Settings and select Manage Sites from the left navigation.
  2. Find the site and select the site's Backend Settings and on the Details tab select the Capillary Web Service Context.

Note. Dependant on the web service context configuration.
