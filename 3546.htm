Display Finishing Touches on Menu Item

To display additional items to select for a menu item:

Step 1. Turn on the setting to display the finishing touches category.

Step 2. Add the finishing touches component to layout for the checkout.jsp and the reviewAndCheckout.jsp pages.

Step 3. Define the Screen label to change the text in this example Finishing Touches is the label.

Frontend Example


Step 1. To enable the Finishing Touches Setting

  1. In Site Manager go to Site Settings > Manage Sites > select the site's Front End Settings go to Online Ordering Settings.

Go to the Finishing Touches Category Id setting and define the category Id corresponding to items considered to be "Finishing Touches". These items display as checkboxes on the site's frontend. 

Save the changes and refresh the site cache. 


Step 2. Add Finishing Touches Component to the Page Layout

  1. In Site Manager, go to the Content section and select Sections & Pages from the left navigation, select the Online Ordering Pages


2. For the checkout.jsp and the reviewAndCheckout.jsp pages, go to the desired page layout or create a new one.


3. To edit a page the page must be in a declined status. Add the Finishing Touches component to the page layout.

Step 3. To Change the Finishing Touches Text Define the Screen Label

  1. The default text displays as Finishing Touches, to change the label to different text in Site Manager, select the Content tab select the site from the Edit Site drop down list.
  2. Select Screen Labels from the left navigation, go to the myres.order.finishingTouches.title
  3. Define the text for each language and select Apply and Save.
