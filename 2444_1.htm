View Logs

To access the View Logs (for webHotel or Mobile Sites) go to the System Tools tab and select the View Logs from the left navigation. The log files are sorted ascending by modified date. See also: Logging Settings

The log files here correspond to ..HCP\webHotel\logs on the server.

Types of Logs:

  • Mobile:
  • webHotel
  • Misc Logs (non-opc logs)

View Logs

  1. Select the webHotel, webMobile Logs, or Misc Logs from the left navigation.
  2. Select View to open the log.
  3. (optional) To search for logs on a specific date, use the calendar to add the date to the date to the search filter.
  4. (optional) Filter: Enter part of the log name, and select Ok to filter the log files by name.
  5. Select Ok.

To Copy the Log

To copy the contents of the log into another file open the log by selecting the View link and then select the text by right clicking to select copy.
