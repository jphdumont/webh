Add or Update Membership Information

To add or update membership information from the Profile option

  1. Select Profile from the OPERA2Go main menu screen. (Alternativly, you can add membership information from the Edit Reservation Screen. Select Due in from the main menu, select the reservation to edit and add membership information under the membership section.)
  2. From the Profile Lookup screen, enter one or more of the following search criteria and select Search: last name, first name, membership no., email or credit card number.

  3. From the Lookup Results screen, select a profile to update.

  4. From the Guest Profile screen, select Membership Information and select Add if a record does not exist.

  5. On the Add Membership screen, complete the below information and select Save.
  • Membership Type
  • Membership No.
  • Membership Level
  • Enrollment Code
  • Expiration Date
