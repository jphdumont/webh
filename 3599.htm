Create a Special Deals Page

To create a deals page in myRestaurant Sites :

Step 1. Create the Generic Deals Page ( deals.jsp )

Step 2. Define the Promo Area on the Page

Step 3. Define Product URLs

Step 1. Create the Marketing Generic Page for the Special Deals

  1. In Site Manager go to the Content tab.
  2. Select the site from the Edit Site drop down.
  3. Select Section & Pages from the left navigation.
  4. Select My Restaurant Web Marketing pages and create a Generic Page. Define a logical URL and page name, for example /myres/deals.jsp.  

Step 2. Create Promo Area on the Page

  1. Customize the page layout of new generic page to include a promo area.
  2. Edit the Promo Area to display the specials and deals. Use rich text content to create the layout using HTML. 
  3. Upload images to use on the promo area and add the links. 
  4. In this example layout a grid containing the images of the deals and specials with link format listed below. 

Step 3. Define Product URLs

To define the link to the deals this format is required: 

<Base Site URL>/myres/landing/menu.jsp?CAT1=<n1>[&CAT2=<n2>]*[&PROD=<pid>] 

URL Requirements:

  • Specify one or more Category IDs, and/or a Product ID. Users should ensure that they provide the correct sequence and values of Category IDs.
  • Make sure the Product ID exists under the specified Category IDs. 
  • If a Product ID is not specified, user is redirected to the menu page showing the list of categories/products under the last Category ID specified in the URL. If specified, the request parameters initiate the Category to be automatically selected. 
  • If a Product ID is specified, user will be redirected to the menu page where the Product is located and the product is pre-selected (expanded). 
  • Category IDs and Product IDs should be provided by the one who configured the Menu of the store. They are the ones who knows which categories / products are deals or specials. 

URL Examples: 

myresl13.webhotel.com/myres/landing/menu.jsp?CAT1=126&PROD=3115 
myresl13.webhotel.com/myres/landing/menu.jsp?CAT1=188&CAT2=221 
myresl13.webhotel.com/myres/landing/menu.jsp?CAT1=188&CAT2=221&PROD=3837 
testsite.webhotel.com/myres/landing/menu.jsp?CAT1=846 
testsite.webhotel.com/myres/landing/menu.jsp?CAT1=846&PROD=8319 

Frontend Example

To see the Deals page on the frontend of the site.

  1. Go to the site.
  2. Select a Store and Order Date and Time. 
  3. Go to the generic Deals page using the new link on the menu page or the URL directly.
  4. Select the image with links to go to the desired category and product. 
  5. The user redirects to the appropriate category page. If a product is defined as a special deal, the product will be pre-selected (expanded).
