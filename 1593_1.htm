Create a Custom Homepage for a Mobile Site

To create custom pages for a mobile site:

  1. Go to the Site Manager under the Content tab, select Sections and Pages then select the General Pages (Mobile) link or Edit.
  2. Now you can create a page or edit an existing page. To create a new homepage for your mobile site: select Home Page from the page type drop down list. Define the page name and the page path, then select the Create a Page button.
  3. To edit select the page and make changes, then Save.
  4. To configure the new page, select page configuration.
  5. To make changes to the page layout or design select edit the layout. Under the Area tab, select save and apply to save your changes.

To define the URL as the new homepage:

  1. Go to Manage Sites > Site Settings > Front End > under the URLs tab in the Custom Splash Page field, enter the path of your new homepage.
    For example: /tmpl/homepage.xhtml
  2. To see your new page, visit the website in a browser.
    Note. If you do not see your changes, clear your browser's cache then reload the page.

Tip. In Site Manager use the Edit Site: drop down list on the top left of the browser to choose which website you want to edit.
