Create a Key Using a QR Code

To Use myKeys

The Guest can scan the QR code on the Reservation Confirmation mobile web page or from (the Guest's) Reservation Confirmation email (from their mobile device). Alternatively, the Guest can also scan the QR code on the printed Reservation confirmation.

To Setup the myKeys Application

  1. Step 1: Configure the Key Encoder Settings

Step 2: Define the Key Encoder Type in Mobile Site Site Manager

  1. To define the key encoder in the Site Manager go to Site Settings > Manage Sites > find the (Mobile) Site > select Frontend Settings > go to the Site Defaults tab.
  2. Select which Key Encoding System will read the generated QR code. If this parameter is set to "None" then the QR code does not display on the Reservation Confirmation page or the Reservation Confirmation email.

  1. None. If this parameter is set to none then the QR code does not display on the Reservation Confirmation page or the Reservation Confirmation email.
  2. MICROS. When this is set to MICROS, then the QR code that is generated will be able to be read by an MICROS Kiosk.
  3. Ariane. Ariane is a third party kiosk solution. When this is set to Ariane, then the QR code that is generated will be able to be read by an Ariane Kiosk.

Step 3: Define the Secondary Validation in Site Defaults (Mobile Sites)

Select to turn on or off the secondary validation.
