Add images for Payment Types on the myAccount Page

To add images to the payment types on the myAcount.jsp page:

Step 1. Upload images to Manage Media

Step 2. Add the Image URl to the Payment Images in the Frontend Site Settings

Step 1. Upload images to Manage Media

  1. In Site Manager go to the Content tab and select Manage Media from the left navigation.
  2. Upload images for each payment type to image server right click and copy the URL from the image uploaded

Step 2. Add the Image URl to the Payment Images in the Frontend Site Settings

  1. In Site Manager go to the Site Setting tab > select Manage Sites > go to the site's Front End Setting > and define the Payment Images.
  2. Use the plus icon to define card types, add the cards that display in the drop down on the frontend of the site.
  3. Add the image URL for the respective credit card type.
  4. Define the Markup Class for each credit card as cc-img.
