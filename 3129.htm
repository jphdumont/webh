Add an Option to Edit My Order on the Sidebar

The configurable order list displays on the Menu, My Order, Review and Checkout, and the Order Confirmation pages. When the component is on the confirmation page it is not editable.


To turn the option to Edit the My Order List:

  1. In Site Manager go to Site Settings > Manage Sites and select the Site's Frontend Settings and select the Display Options tab.
  2. On the Display Options tab turn on the "Display Customized Items on Item Basket", when this parameter is off, the modifiers for a item will be hidden.
