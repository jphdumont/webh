Add Sites to a User

In order for a user to be able to access the site the site needs to be assigned to the user.

To add a Site to an Existing User:

  1. Log in to Site Manager with a System Administrator access rights.
  2. Go to Manage Users and select Users for the left navigation.
  3. Next to the user name select edit.

4. Go to the Associations tab.

5. Assign the sites to the user by selecting the sites in the Not Assigned text box then select the Add button to move the sites to the Assigned Sites text box.
