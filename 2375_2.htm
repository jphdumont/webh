Use a Dynamic Variable to add a Direct Link to the Guest Reservation

Using a dynamic variable create links directly to the Guest Reservation These links can be used to bring the user directly to their reservation. For email templates, in webHotel, or in webMobile use these URLs to link to the Guest Reservation by inserting the dynamic variable ${reservation-check-params} in the href tag.

To Define the encoded URL to pull up the Guest Reservation Directly from Email Templates

For example this can be used in the email templates for Confirm, Modify and Cancel Reservations. To use in the email templates go to the Content tab and select the Email Templates from the left navigation. Then edit HTML Body Contents to include encoded URL to go directly to reservation.

To make a hyperlink to the Guest Reservation use the syntax below.

For webHotel sites use this example:

<a href="https://${host-url}/bp/view_reservation_detail.cmd?${reservation-check-params}"> View Reservation </a>

For Mobile sites use this to link directly to the Guest Reservation Details :

<a href="https://${host-url}/mobile/faces/tmpl/msm/resv/reservationLookupPage.xhtml?${reservation-check-params}"> View Reservation </a>
