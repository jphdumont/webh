Add the Ability to Save & Send During the Booking Process in webHotel

During the process of making a Reservation, this feature provides the ability to Save and Send by email the Booking details with all the available packages and rates available at the time the user was making the booking. When this feature is turned on an additional Save Booking Email Template is available.

To implement the ability to Save & Send Booking functionality:

Step 1. Turn on the Display Option > Email Guest Booking Selections

Step 2. Create and Modify the Email Template for Save Booking

Step 3. Change the Screen Label for the button (optional)

Save and Send is available on these pages during the Booking Process:

  • Search Results page Page: /bp/search_regional.cmd
  • /bp/select_package.cmd
  • Enter Guest Details Page: bp/select_package_book_room.cmd

Step 1. Turn on the Save Booking Selections Button in Display Options

By default this feature is turned off.
To turn this feature on:

  1. In Site Manager go to Site Settings.
  2. Select Manage Sites from the left navigation.
  3. Select the Site's Frontend Settings.

  1. Select the Display Options tab.
  2. Turn on the Email Guest Booking Selections to enable.

Step 2. Create and Modify the Email Template for Save Booking

  1. In Site Manager go to the Content tab
  2. Select the Site from the Edit Site drop down list.
  3. Select Email Templates from the left navigation.
  4. Select the Save Booking Email Template from the drop down list and select the button to Create Email Templates.


To Add a link to View the Saved Booking Details:

To add a link to the view the Saved Booking Details use this variable to the link:

Saved Booking Details: <a href="${link-url}">Select to view your save booking details</a>

Note. Create the email template for each property if it is a chain site.
