Add Analytics to Multi-Language Pages (Mobile)

Using multilingual pages and including IBEs, by setting up the Section & Pages as multilingual. Some analytic providers have tracking codes that are language specific. When using the multilingual pages enter a analytics code for each language. In Site Manager, to add analytics for pages in multiple languages go to the Content tab and select Sections & Pages from the left navigation menu.

Note. This functionality is only available when the multi-lingual checkbox is selected. If the select a language drop down is not displayed make sure the multi-lingual checkbox is selected next to the page on the Sections & Pages section of Site Manager. Clear Site Manager and browser cache to see the changes.

Add an Analytics Provider for Each Page in the Different Language

The analytics providers are turned off by default.

To add analytics per page:

  1. In Site Manager go to the Content tab and select Sections & Pages from the left navigation.
  2. If the Multilingual checkbox is selected next to the page, then select the Page Configuration and go to the Analytics tab.

  1. Select the language from the drop down list and add the language specific analytics code per page.
  2. Select Add a Provider to add another analytics provider. Or select Edit next to an existing Analytics provider to add analytics for this language to the multilingual page.
  3. Select Save to save the Analytics to the page.

  1. Add the analytics for each language and select Save Details.
