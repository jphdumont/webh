Audit Records

The Audit Records report displays an audit trail of actions that were performed by Site Manager users.

Start Date. Enter a date, or select the button to choose a date, to begin the report results with.

End Date. Enter a date, or select the button to choose a date, to end the report results with.

Record Type. As Show All is selected by default, select the down arrow to choose a specific record type to view the trail of actions performed for.

User Name. As Show All is selected by default, select the down arrow to choose a specific User Name from the drop down list.

Entity ID.

Format. Select the output format (PDF, CSV, Image) for the report.

Site. Select the property to run the report for from the list in the drop down list. The properties that are available for selection will be based on the properties that the user has been given permission access to.

Function Buttons/Links

OK. Select to run the report and obtain the output based on the report criteria that was selected.
