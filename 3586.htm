Setup a Unique Pages for each Property Information on a Chain Mobile Site

On a Mobile chain site, selecting property information option from the frontend of the site displays the list of properties in the Hotel chain. Configure unique pages for each property information page tab bar component section of the Mobile site. Use this feature to setup unique property info details, directions, attractions, restaurant information, and photo galleries unique for each property of the chain site.

To setup a photo gallery for each property on a mobile chain site:

Step 1. Schedule Content on a Marketing Pages for each of the Tab buttons

Step 2. In Manage Site Elements Associate the Unique Marketing Pages with the Tab Bar for Each Property

Step 3. Configure Screen Labels for the Tab Component on the Mobile Sites

View Frontend Example Screenshot

Step 1. Schedule Content on a Marketing Pages for each of the Tab buttons

  1. In Site Manager, go to the Content tab and select the site from the Edit Site drop down list.
  2. Select Sections & Pages from the left navigation > select Edit next to the Marketing Pages.

  1. Add a New Layout or Edit existing page layout > and add the system component Tab Bar.
  2. Add a Promo Area (ex: Promo E)
  3. Schedule new content for this promo area > (for example) add a photo gallery

Step 2. In Manage Site Elements Associate the Unique Marketing Pages with the Tab Bar for Each Property

  1. On the Content tab, select the site from the Edit Site drop down list.
  2. Select Manage Site Elements from the left navigation, and select Property from the Type drop down list.
  3. Select Find.
  4. Select Edit next to the Property. Each property for the chain site is listed here.

  5. Select the Attributes tab, select a marketing page from the drop down list for each tab option on the frontend of the site.
  6. Repeat Steps 1-5 for each property of the chain site.

Step 3. Configure Screen Labels for the Tab Component on the Mobile Sites

To Configure the screen labels for the tab component on the Mobile chain site:

  1. In Site Manager go to the Content tab.
  2. Select the Site from the Edit Site drop down list and select Screen Labels from the left navigation.
