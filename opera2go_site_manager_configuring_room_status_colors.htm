Configuring Room Status Indicator Colors

You can configure the colors used for room statues throughout the OPERA2Go application. For your OPERA2Go site, select Motif Management>Application Colors>Generic Components>Room Status Indicator to configure the room status color settings for the site template.

In OPERA2Go, the below room statuses are supported.

Clean

Dirty

Inspected

Pickup

Out of Service/Out of Order

To configure the colors used for the room status indicators

  1. In Site Manager, select the Motif Management tab.

  2. From the Edit Site drop-down list, select an OPERA2Go site.

  3. Select the Application Colors link on the Motif Management pane.

  4. Under "Site Categories," expand the navigation tree for the Generic Components category to view its sections.

  5. Under the "Generic Components" category, expand the navigation tree for the Room Status Indicator section.

  6. Under the "Room Status Indicator" section, select the room status indicator for which you want to change the background and text colors.

  7. From the color picker tool, select the color you want for the room status indicator. You can select the color using one of the following methods:

    Selecting a location in the colored pane. Use your mouse to move the selector around the colored pane until you find the desired color.

    Using the arrows on the color spectrum bar to change the color spectrum. Use your mouse to slide the arrows up and down the spectrum bar until you select the desired color.

    Modifying the individual RGB values by manually entering values into the fields.

    Entering the HEX (hexadecimal) value for the color.

  8. Select the Save Colors button to apply the color settings.

To restore the original colors of the OPERA2Go site's template:

  1. Select the Restore Original Template button.
