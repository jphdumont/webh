Configure Exact Target Email Integration for myRestaurant Sites

In myRestaurant Sites integrate Exact Target to send email signup and account information to exact target for email distribution.

Setup in Exact Target Integration in Site Manager:

Step 1. Enter the Display Options tab and input exact target client id- and list Id

Step 2. Add the Display Email Club Sign Up Component to the Create Account Page and the Check-Out Page

Step 1. Enter the Exact Target IDs in the Display Options

  1. In Site Manager, go to Site Settings > Manage Sites and select the site's Frontend Settings.
  2. Select the Display Options tab and enter the exact target Client Id and Exact Target List Id

Step 2. Add page component to the Page Layout for Display Email Club Signup

  1. In Site Manager go to the Content tab and select Sections & Pages

To Test the Exact Target Integration

Verify in the Logs the Exact Target Integration

In the log file , find "EXACT TARGET RESPONSE:", the status is "OK" to verify the API integration is successful.
