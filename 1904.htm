Setup Dining Reservations for a Mobile Site

Provide the option for a Guest to log on to your Mobile Hotel website and make a Reservation at one of your Hotel's Restaurants. The Guest will specify the Reservation details and their contact information. On confirmation, an email will be sent to the Dining Reservation Request Recipient Email(s), and copied to the specified email address of the Guest.

Step 1. Turn on the Dining Menu Button on the Mobile Homepage:

  1. In Site Manager, go to Content tab > select Mobile site to edit (from the top left corner) > select Sections and Pages > select General Pages (Mobile) > select the Component Configuration column for Home Page entry > search for msm.homePage.diningBtn and set Render value to true.
    See Also: Add a Custom Menu Display for a Mobile Site Homepage

Step 2. Configure Restaurants per property:

  1. In Site Manager, go to Content tab > select your Mobile Site to edit > click Manage Site Elements.

    Note. Restaurant information can be imported from OPERA into MCP using the Hospitality Product Import functionality or added in MCP directly.
    See Also: Hospitality Products

  2. To add a restaurant, to go to Add an Element select Create New and specify details with Type = Restaurant and select a Property.
  3. To check available Restaurants, in Find a Site Element tab, from the Type drop down list select Restaurant. Then select Find to see which Restaurants are already setup.

Tip. In Site Manager make sure you have the correct property selected in the top left drop down menu.

Step 3. Configure Dining Reservation Request Recipient Email(s):

The email address is configured per restaurant and multiple email addresses can receive the reservation request email.

  1. In Site Manager, go to Content tab and select Manage Site Elements, select the type:Restaurant and go to the Attributes tab.
  2. Specify the value Dining Reservation Request Recipient Email(s). This is a comma delimited list of email addresses to which the dining reservation request will be sent. Syntax= Name <emailAddress>, For example: John Smith <jsmith@micros.com>

Step 4. Go to your Mobile Site and Test

  1. From your mobile site, select the Dining option from the main menu and select the Restaurant > enter the date, time and contact details.
  2. Select Confirm.
  3. Confirm the email recipient receives the email with the Dining Reservation details.
