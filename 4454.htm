Text Field Auto Complete Pickup Address

Use the Select Location by single box pickup address component to enable an auto complete search for the pickup address lookup text field on the search locations (searchlocations.jsp) page:

  • Prerequisite: It is required to have online ordering activated. See Also: Enable Online Ordering
  • Step 1. Turn on the Display Options parameter Support for Single Box Pickup Address
  • Step 2. Use the Select Location by single box pickup address component on the searchlocations.jsp page

Frontend Example : Select Location by single (text) box to input the pickup address

On either search location page using the Select Location by single box pickup address component on the page layout.

Step 1. Turn on the Parameter in Display Options

  1. To turn on the parameter to enable the functionality in Site Manager go to the site's frontend settings to turn on the parameter to enable Support for Single Box Pickup Address.
  • When the parameter is "On", we will show single box for pickup address and uses address validation to suggest street address.
  • When the parameter is "Off", then on the frontend when searching for the pickup address there are no auto complete suggestions.


Step 2. Add the component to the page layout

To add the Select location by single box pickup address go to the Search Location (searchlocations.jsp) page and select to edit the page layout and add the component to the custom page layout.

  1. In Site Manager go to the Content tab.
  2. Select a site to edit from the edit site drop down list.
  3. Select Sections & Pages from the left navigation.
  4. Select Edit next to the My Restaurant Web Online Ordering Pages from the Sections \
  5. Select Edit next to the Search Locations Results Page or the Search Locations Criteria and results page.

  6. Select Edit and Edit Page Layout.
  7. Create a Grid layout.
  8. Add the Select Location By Single Box Pickup Address to the page layout. (Required. It is required to include the Select All Locations Result component on the page layout.)
  9. Apply and Save Layout.
