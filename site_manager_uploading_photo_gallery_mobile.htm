Uploading a Photo Gallery for a Mobile Site

Photo galleries can be uploaded for mobile sites under Content>Sections & Pages>General Pages>Edit by selecting the Edit icon under the Page Configuration column.

To upload a photo

  1. Go to Site Manager>Content>Sections & Pages.
  2. From the Edit Site drop-down list, select a mobile site.
  3. From the Site Sections grid, select General Pages.
  4. Select the Edit icon under the Page Configuration column for the "propertyInfoPhotos" row.
  5. Create a new photo gallery for this page by selecting the Edit icon for the area on the page where you want to upload the gallery.
  6. Under the Areas tab, select the Asset Type down arrow and choose an Image from the drop-down list.
  7. Select the Format down arrow and choose a format type for the image.
  8. Under Asset Options, select the Add Image button and select and an image file from the Media Browser window.
  9. Finish adding the image by entering the following properties for the image and selecting the Save Properties button: ALT Name, Width, and Height. Additionally, you may also enter a URL to the image (select the URL radio button and enter URL) or choose not to link it by selecting the Image not linked radio button.

To verify the upload was successful, access the mobile site and go to the property photo gallery page. The uploaded photos should display in this area.
