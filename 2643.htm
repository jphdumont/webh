OPERA2Go Accessing Sales and Catering BEOs

In OPERA2Go view the Sales and Catering Banquet Event Order (BEO).

To access Sales and Catering BEOs (Banquet Event Orders)

  1. From the OPERA2Go Main Menu select the Search BEO button.
  2. Enter a start and end date for the BEO (Banquet Event Order).
  3. Select Generate Report to view the BEO.


    Select to open the BEO pages preview. Click the pages to access more pages of the BEO.
    Navigating around the BEO pages: Scroll down the page to see all the BEO pages.

Turn on the option for the OPERA2Go Main Menu

  1. In Site Manager go to the Content tab and select the Pages & Sections from the left navigation.

2. Access the Component Configuration to turn on and off the icon from the OPERA2Go main menu (homepage).

3. Change the Render value to false to turn the search BEO option off from the Main Menu.

To add an image for the Search BEO button

  1. In Site Manager go to Motif Management and select the Application Images.
  2. Select the Group Name Home page / Menu Dialog Button Icons.
  3. Select edit and upload the image or select the image from Media Manager.
  4. Save the changes.
