View a Guest Folio

To view a Guest's Folio:

Method 1

  1. From the Dashboard screen, select Due Out.
  2. From the Lookup Results screen, select the View Folio icon.

  3. The Guest's Folio will display with the Total amount listed at the bottom.

Method 2

  1. From the Dashboard screen, select Reservation.

  2. From the Search screen, enter either a Last Name, First Name or Confirmation Number and select the Search button.

  3. Locate the Guest and select the View Folio icon.

  4. The Guest's Folio will display with the Total amount listed at the bottom.
