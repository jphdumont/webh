Adding Buttons and Icons to a Mobile Site

To add or change the icons on a Mobile site go to Motif Management > select Application Images from the left navigation and then select Buttons from the Image Categories. The buttons and icons are organized into Groups. To narrow the list select a Group Name from the drop down list.

To Change the icons on the Homepage of a Mobile Site:

Step 1. In Motif Management > select Application Images > and select Buttons from the Image Categories.

Step 2. Select the Home Page / Menu Dialog Buttons Icons from the Group Name drop down list.

Step 3. Find the button and select the Edit icon.

Step 4 - 6. Upload the image to the Media Manager, you must select the Apply Button. To add a new image you must Clear the Image in order to Apply a new Image.

Step 7. Select Save button.

Note. On the Mobile site, the size and name of the button can be viewed in the page source.
