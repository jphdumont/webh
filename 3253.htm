Add the Edit your Profile Component to a Page

To add the Edit your Profile Component to a myRestaurant webpage add the component primary contact info:

  1. In Site Manager select the Content tab.
  2. Select Pages & Sections from the left navigation.
  3. Select myRestaurant Web Account Pages, then Edit the My Account Page, Edit, add the component Primary Contact Info to the page layout.
  4. Apply and Save.
