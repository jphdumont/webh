Enable Last Store Saved in Browser Cookie

To enable the user's last store saved in the browser's cookie turn on the parameter and add the component to the page layout.

Step 1. Turn on the Display Options Parameter Store Selected Location in Cookie

  1. Go to Site Settings > Manage Sites and select the site's Front End Settings.
  2. Go to the Online Ordering Settings tab and turn on the Store Selected Location in Cookie parameter.

Step 2. Add the My Last Store Component to the Search Locations page

  1. Go to the Content tab and select the site from the Edit Site drop down list.
  2. Select Sections & Pages from the left navigation. Select Edit next to the Online Ordering Pages
  3. Select Edit next to the Search Locations page > and add the My Last Store component.
