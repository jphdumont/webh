Create a New Folder

To create a new folder:

  1. Navigate to the folder that will be the parent of the new folder.
  2. Click the right mouse button. A pop-up menu will appear.
  3. Select "Create" in the pop-up menu. A new folder will be created under the selected folder.
  4. Enter the name of the folder.
  5. Select "Enter".
