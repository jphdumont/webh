Working with Page Layouts

To access this feature, in Site Manager go to the Content tab, then select Sections & Pages from the left navigation menu, find groups of pages with the page you would like to edit. Select the Edit icon next to the page you would like to edit then select Edit Layout button to access the Layout Details page.

To create a new page layout

  1. On the Layout Details page, select the Layout tab to create new layouts for the page by dragging and dropping layout blocks onto the page.
  2. In Site Manager, go to Content>Sections & Pages and select the edit icon for the page section you wish to edit.
  3. On the Pages tab, locate the page you wish to edit and select the edit icon.
  4. On the Areas tab, select the Edit Layout button.
  5. Under the Layout tab, left-click and hold a layout block and drag and drop it onto the desired page area. The available page areas will be highlighted on the grid.

Tip: You can remove a layout block from the page by selecting the delete icon in the upper right-hand corner (see screen below). You can also change the height of the block by selecting the edit icon (see screen below).

Customize the page layout with content areas

On the Layout Details page, select the Areas tab to add content placeholder areas to the page layout.

  1. In Site Manager, go to Content > Sections & Pages and select the edit icon for the page section you wish to edit. On the Pages tab, locate the page you wish to edit and select the edit icon.
  2. On the Areas tab, select the Edit Layout button and on the Layout Details screen, select the Areas tab.
  3. Under the Areas tab, left-click and hold a content area and drag and drop it onto the desired layout block.
  4. Select the Save Layout button to save the content areas that have been added to the page.

Tip: You can edit a content area by selecting the edit icon in the upper right-hand corner of the content area (see screen below). The Area Settings screen will open. You can delete a content area by selecting the delete icon in the upper right-hand corner (see screen below).

Create a custom content area

On the Layout Details page, under the Areas tab, select the Add Area button to create your own custom content area.

  1. In Site Manager, go to Content>Sections & Pages and select the edit icon for the page section you wish to edit.
  2. On the Pages tab, locate the page you wish to edit and select the edit icon.
  3. On the Areas tab, select the Edit Layout button and on the Layout Details screen, select the Areas tab.
  4. Under the Areas tab, select the Add Area button. The Area Settings screen will open.

  5. On the Area Settings screen, complete the below fields and select Ok. Select the Save Layout button to save the content areas that have been added to the page. See Layout Details for more information on the Area Settings screen.

    Name. Enter a name for the area.

    Description. Enter a Description for the area. The Description is mandatory.

    Width. Define the width select from pixel unit, automatic.

    Height. Define the height select from pixel unit, automatic.

    Alignment. Select the alignment for the component.

    CSS Selector. Define the CSS class name.

    Unique Content. Check this box to mark the content as unique content. This content will not appear on other pages, it will be unique to this page area.

To Copy a Page Layout to Another Page

  1. Go to the Content Tab, select the Sections & Pages.
  2. On the Layout Details page and select the Areas Tab, then select the Copy Layout button.
  3. On the Copy Layout dialog box select the page to copy to from the drop down list.
  4. Select Ok.

To Copy Content Area Settings to Another Page

  1. From the Areas tab select the blue icon in the on the top right of the predefined content areas (see the example below).

  2. Then the Area Settings dialog window appears, select the Copy Settings button.

  3. Then select the page to copy the settings to and select Ok.
