Timeframe

Enter a start and or end date for the targeted content. If no start or end date is specified, the start date will be immediate and end date will be indefinite.

Start and end date cannot precede or exceed that set for the experience. Adjust the experience timeframe if you would like to schedule this targeted content timeframe outside of the start and end date scheduled for the experience.
