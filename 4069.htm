Add Use my Location Button to the Search Locations Page

To add the Use My Location Button to the Search for Restaurants page:

Frontend Example of Use My Current Location

  1. In Site Manager go to the Content tab.
  2. Select Sections & Pages from the left navigation and select the Online Ordering Pages.
  3. Edit the layout for the restaurantSearch.xhtml page.
  4. Add the Component Use Current Location button to the page layout.
