Add Lightbox of Property Photos on the Regional Availability Page (Mobile Site)

In a Mobile Hotel Chain Site when a Guest searches the Regional Availability page ( bp/search_regional.cmd) there is the option to add a lightbox of up to 10 images per property.

To Configure Lightbox on Regional Availability Page:

  1. In Site Manager go to the Content tab, select the webHotel site from the Edit Site drop down list.
  2. Select Manage Site Elements from the left navigation.
  3. Select Property from the Type drop down list, select Find button.
  4. Select Edit next to the property to add the images.

  1. Go to the Images tab and add up to 10 images.

  1. Go to the frontend of the site and test the lightbox display on the Regional Availability Page ( bp/search_regional.cmd).
