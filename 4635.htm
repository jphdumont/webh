Horizontal Accordion Checkout Details Component

To display the 4 step (My Order, Checkout, Details, Payment) accordion checkout process add the Horizontal Accordion Checkout Details Component to the Review and Checkout Page template.

Step 1. Add Horizontal Accordion Checkout Details Component to the Page Template

To add the component to the page template:

  1. In Site Manager > go to the Content tab > and select Sections & Pages from the left navigation.
  2. Select Edit next to the Web Online Ordering Pages
  3. Then select Edit next to the Review and Checkout Page > select Edit Page Layout and add the Horizontal Accordion Checkout Details component.

Optional Step. To Use the Checkout Accordion on the Page Template Use this Shortcode

{Horizontal Accordion Checkout Details}
