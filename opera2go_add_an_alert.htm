Add an Alert

To add an alert to a reservation

  1. From the Dashboard screen, tap In House.

  2. From the Lookup Results screen, tap the Special Request icon.

  3. From the Special Request screen, tap the Alert section to expand it and tap the Add button.

  4. From the Edit Alert screen, select an Alert Code, Alert Area (e.g., Reservation, Check-In, Check-Out) and enter a display sequence for the alert. Enter a description for the alert.

  5. Select either Screen Notification or Printer Notification for the alert.
  6. Tap the Save button.
