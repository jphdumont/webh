Responsive myRestaurant Template

To access the responsive design template:

  1. In Site Manager go to Motif Management select Templates from the left navigation.
  2. Select the site from the Edit Sites drop down list.


To activate the template select the radio button to the left of the template and select activate template.
See Also: Templates


To customize the template edit the CSS, See Also: Edit the CSS for a Template
