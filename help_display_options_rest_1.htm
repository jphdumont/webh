Display Options (for Restaurant Sites)

To access in Site Manager go to the Site Settings tab and select the Site's Frontend Settings, then select the Display Options tab.

Note. Clear the Cache after every change to see the update on the frontend of the site. Clear the browser cache and the site cache.

When this parameter is on a customize button displays next to the Add to Cart button, and the page redirects to the customization page. The non-mandatory modifiers are only accessible on the step by step customize page which users can access by clicking the Customize Button.

Use Multi-level Modifiers.

Turn this parameter on to enable multi-level modifiers, for example when the product (item) has multiple levels of customizations (containers and modifiers). Depending on the myCentral configuration on each container, the options under each container will show either a radio or a checkbox or a select control. When a checkbox or a radio button is selected and it has a sub container, it will show the sub container (initially hidden). A radio button displays if the container is mandatory and the quantity is 1, otherwise a checkbox displays with the customization options.

View Popup when selected store is offline

If a Store does not have online ordering active, use this feature to offer the ability for the user to browse the menu then (if configured) the system displays a dynamic phone number for the selected location.

To turn this feature on in Site Manager:

  1. Go to the Site Settings > Manage Sites > choose the Site's Front End Settings, and select the Display Options tab.
  2. Turn on parameter Show Popup Error Message on Menu Page when Selected Store is Offline

Frontend Example

When the user selects the location that is offline, then the popup message displays offering the phone number pulled dynamically from the Store Details section.

 

To Edit the Popup message in Screen Labels

  1. Go to the Content tab and select Screen Labels from the left navigation.
  2. Select the label group: My Restaurant Web Menu
  3. Find the Label : myres.order.menu.lbl.storeOffline and edit the message.

    Enter any of these Dynamic Variables in the popup message:

    ${storeTelephone}

    $ {storeId}

    $ {storePrimaryAddress}

    $ {storeSecondaryAddress}

    $ {storeLong}

    $ {storeLat}

     

To Configure the Store to be Offline

  1. In Site Manager go to the Content tab and select the Manage Site Elements.
  2. Select Restaurants from the Type drop down list.
  3. Select Edit next to the Restaurant to make changes.
  4. Go to the Attributes tab and enter N in the Online Ordering field.

Enable Dynamic Subtotal

When this is set to off the Subtotal displays the total of all item prices in the basket. When this is set to on the Subtotal should not display on the Totals and Order Status 2 components unless one of these is present:

  • Coupon Discount
  • Delivery Charge
  • Tax
  • Holiday Tax

The Subtotal and Grand Total (Your Total) should never display together alone.
