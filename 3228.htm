Working with the Account Login Page for myRestaurant Sites

Creating the login page for myRestaurant sites there are 2 options for the Account login form the default login component and the customized login component.

Default login component. The default login component provides a forgot password link.

Frontend Example:

Customized Login Form. The customized login form provides the option to auto generate a system password or reset with an email link.

Frontend Example:

To Add the Login Form to a Page

  1. In Site Manager, go to the Content tab and select the site to edit from the Edit Site down list on the top left.
  2. Select Sections & Pages from the left navigation, select the myRestaurant Web Account pages and select the Login page (or another page).
  3. Select Edit Layout
  4. Add the Customized Login Form or Login Form to the page layout.
