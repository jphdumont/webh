Create New User

In order for someone to access Site Manager they must be setup as a user and associate the site they want access to the user. To access the user rights area of Site Manager, select the Manage Users tab from the top navigation.  

To create a new user:

  1. Enter a first name for the user in the "First Name" text field.
  2. Enter a last name for the user in the "Last Name" text field.
  3. Enter the user name of the user's Opera login in the "Existing Opera Username" text field. When clicking on the "?", a message tip is displayed stating that the Opera username entered must match exactly to the current Opera username and will be validated when submitting the site setup form.
  4. Enter the password of the user's Opera login in the "Existing Opera Password" text field. When clicking on the "?", a message tip is displayed stating that the Opera password entered must match exactly to the current Opera password and will be validated when submitting the site setup form.
  5. Re-enter the password of the user's Opera login in the "Confirm Existing Opera Password" text field.
  6. Enter the Site Manager login name for the user in the "WebHotel Administrator Username" text field. Selecting the "?" displays a message tip to notify the user that the login name must be an email address less than 80 characters. It does not need to match the Opera login name.
  7. Enter the Site Manager password for the user in the "WebHotel Administrator Password" text field. Selecting the "?" displays a message tip to notify the user that the password must contain at least eight characters with no spaces, and must contain at least one alphabetic character [a..zA..Z], one number [0..9], and one character from this list: `!@$%^&*()-_+=[];:,<.>/?.
  8. Re-enter the Site Manager password for the user in the "Confirm WebHotel Administrator Password" text field.
  9. Select Create User. The page will refresh and you will see tabs for "Details," "Associations," "Roles," and "Role Groups."

  10. On the "Details" tab, select or deselect the Active checkbox to make the user active or inactive.
  11. Select the Associations tab.

  12. In the "Not Assigned" list box, select the site(s) the user will be able to make modifications to.
  13. Select the Add button to move those sites to the "Assigned Sites" list box.
  14. Select the Roles tab.

  15. In the "Not Assigned" list box, select the Role(s) for the user in Site Manager.
  16. Select the Add button to move the Role(s) to the "Assigned Sites" list box.
  17. Select the Role Groups tab.

  18. In the "Not Assigned" list box, select the Role Group(s) for the user in Site Manager.
  19. Select the Add button to move the Role Group(s) to the "Assigned Sites" list box.
  20. Select Back to Users List to return to the list of users. You will see the new user included in the list.
