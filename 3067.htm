Define the Room Image Size on Search Results Page

On the Search Results Page the default image size is 189 px and height 106 px, to change the size of the image on the Search Results Page:

Step 1: Turn on the show rooms with images parameter and define the room image size

  1. In Site Manager > go to Manage Sites and select the Front end Settings for the site > Display options > Display Availability By and select Rooms with Images
  2. In Site Manager > Front end Settings > Display options > enter the desired image size Search results Page Image Size (pixels)

Step 2: To setup the Search Rooms Page to show rooms with images

  1. In Site Manager > go to the Content tab > go to Manage Site Elements
  2. Select Room Type from the Site Elements Drop down list.
  3. Select the Attributes tab to add images for that Room Type.
    Note. Add up to 10 images can be added per Room Type.
