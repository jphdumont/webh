Add Order Time and Location Details to the Order Flow Process

To build the order time and location confirmation details into the ordering process flow use the Order Time Message and the Ordering From component to display the confirmation of the order time and location. The ordering form component includes the ability to change the location and view directions to the selected location.

Frontend Example

Steps to Customize in Site Manager

Step 1. Site's Frontend Settings turn off the Combined Order Summary and Checkout parameter.

  1. In Site Manager go to Manage Sites and select the Site's Frontend Settings.
  2. Go to the Display Options tab and turn off the Combined Order Summary and Checkout parameter.

Step 2. Add the Components to the Page Layout

  1. In Site Manager go to the Content tab.
  2. Select the Site from the edit site drop down list.
  3. Select Sections & Pages from the left navigation.
  4. Select Edit next to the myRestaurant Web Online Ordering Pages.
