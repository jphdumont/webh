Configure the Location Search Page Options

On the frontend of the site, if the user does not choose a store location then the national menu displays without prices. Once the user selects a location then the menu items with prices display.

There are several options to combine the components to create variations of the user experience:

  • Text Field Auto Complete Pickup Address
  • Activate Online Ordering for Now or Later
  • Enable Auto Complete Delivery Address on the Location Search for myRestaurant Sites
  • Add Use my Location Button to the Search Locations Page
  • Configure See Nearby Locations on the LocationDetials.jsp

Use either the Search Locations Criteria and Results Page or the Search Locations Results page.

It is required to include one of each of the 2 groups of components to create a Search locations page:

  1. Search Input Form:
  • Search Locations by Current Location
  • Select Location by single box pickup address (auto complete input text field)
  • Search Location by Single Box delivery Address (auto complete input text field)
    1. Display Search Results:
  • Search Location Result
  • Search All Location Results
