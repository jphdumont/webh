Simple Account Details Component

To display just the logged in customer name, phone number, and email address use the Simple Details Component.

To apply this component to the page:

  1. In Site Manager go to the Content tab.
  2. Select the Site from the Edit Site drop down list.
  3. Select Sections & Pages from the left navigation.
  4. Select Edit next to the page and Edit the Page Layout for the page.
  5. Add the Simple Account Details Component to the page layout.
