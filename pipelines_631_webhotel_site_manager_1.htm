Pipelines

Use this page to view all of the pipeline sources that are configured and currently active. The number of objects within each pipeline pool is also displayed for each source.

To get more information on any pipeline:

  1. Choose the pipeline you are interested in.
  2. Select the "Pipeline Source Name" of the pipeline or select the "View Details" icon. Detailed information about the pipeline will be displayed when the page refreshes.
  3. To return to the list of pipelines, select "Back to Pipeline List".
