Edit or Remove Email Templates

To edit an email template:

  1. Use the "Page Language" drop-down list box to select the language of the template you wish to edit. Templates in that language will be displayed in the lower portion of the screen.
  2. Select the "Edit" icon to the right of the template you wish to edit.
  3. Enter a new subject for the email in the "Subject" text field.
  4. Enter a different source for the email in the "From" text field.
  5. Select the "HTML Body Contents" tab header to modify the HTML version of the email. You may use HTML markup tags to format the content. If you wish to use email tokens in the body, see a list of available tokens here.
  6. Select the "Text Body Contents" tab header to modify the text version of the email. Do not use any HTML markup tags for the text content. If you wish to use email tokens in the body, see a list of available tokens here.
  7. Select "Preview" to preview your changes.
  8. Select "Save" to save your changes.

To remove an email template:

  1. Use the "Page Language" drop-down list box to select the language of the template you wish to edit. Templates in that language will be displayed in the lower portion of the screen.
  2. Select the "Delete" icon to the right of the template you wish to remove.
