Servers

Working with clearing the site cache

When making changes in Site Manager it is required to Clear Cache to make changes display on the frontend of the site. Adding Servers in System Tools > Servers enables the clear cache to apply to all the servers listed as active.

Create Server Information in Site Manager

  1. In Site Manager > go to System Tools > and select Servers from the left navigation to view list of Application servers and Media servers.
  2. To add a new server select the Add Server button.

Server Name. Enter the server name.

Host Name. Enter the host name.

Protocol. Define the protocol, http:// or https://

Port. Define the port.

Server Type. Choose the type of server Application or Image Server.

Active. Check this box if the server is active.

Save Server. Select to save the changes.

Clear Cache Options in Site Manager

There are 2 options to clear the site cache in Site Manager:

  • System Tools > Cache Stats > Select the site and Clear the Cache
    See Also: Cache Stats

    OR
  • In Content use the Clear Cache button in the top right corner. This is site specific, first choose the site from the Edit Site drop down then select the Clear Cache button.
