Localized Labels

The Localized Labels section allows you to update the webHotel database with translated labels for the pages of the webHotel site.

To upload new labels:

  1. Use the "Choose site" drop-down list box to select the site that you want to upload new labels for.
  2. Select "Load Labels". The labels will be uploaded.
