Activate Online Ordering for Now or Later

To activate the option to place an order for the next available (current time) or the option schedule the order for a later time on the frontend of the site. To enable this feature add the Search All Locations Component to the page layout. To configure other options for the search locations page see also: Configure the Location Search Page Options

Note. This functionality is dependant on online ordering being activated for the site. See Also: Enable Online Ordering
This functionality requires the Search results display component also being on the page.

Step 1. Add the the Select All Locations Result component to the Search Locations Results Page

Step 2. To Customize the button text go to Content > Screen Labels

Frontend Example: Display the option to Order Later or Order Now (ASAP order)

When the option to order later is selected the date and time selection is available to select before placing the order

Step 1. Customize the page layout for search locations

Add the the Select All Locations Result component to the Search Locations Results Page or the Search Locations Criteria Page:

  1. In Site Manager -> go to the Content tab > Select a myRestaurant site from the Edit site drop down > Sections & Pages > Create and Edit the layout of Search Locations Results Page using new component "Select All Locations Result" instead of "Search Location Result"

    Note. Please note the "Search Location Result" component does not

  2. It is required to add any of the Search locations Criteria and Search Results Components to the page.
  3. Site Manager > Content > Sections & Pages > Online Ordering Pages > Search Locations Criteria and Results Page or Search Locations Results Page > Search All Locations Results component.

Step 2. To Customize the button text go to Content > Screen Labels

To change the text on the order buttons:

  1. In Site Manager go to the Content tab.
  2. Select the Site to edit from the Edit Site Drop down list.
  3. Select Screen Labels from the left navigation.
