Configure the My Account Page

In webHotel and Mobile Hotel sites when the user logs into their account on the frontend of the site, they go to their account page to:

  • Manage User Name and Password
  • View Reservations
  • Manage Preferences
  • Manage Profile
  • Manage Membership

To Configure the My Account page:

  1. In Site Manager go to Site Settings > Manage Sites and select the Site Frontend Settings Sites > select the Membership tab.
  2. To configure the membership options on the My Account Dashboard page in the Site's Frontend Settings, go to the Membership tab.
