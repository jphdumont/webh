Integrating Third Party Widgets into HCP

Integrating Third Party Widgets into the MICROS Commerce Platform

  1. In Site Manager: Go to Content > Sections & Pages and upgrade the Rich Text Editor. Do the same under Content > Manage Site Elements. This will allow you to add an IFrame widget from the existing Rich Text asset.
  2. In Site Manager: Go to Content > Section & Pages. Under any of the existing areas, a new content type named “Widget Panel asset" will be available with predefined elements. This allows code to be inserted automatically into the existing areas, based on the format selected (IFrame or Inline).

    Note: Currently, only IFrame formats are supported.

    Inline script widgets code can be added through existing Text or Rich Text assets, which do not require a URL option.

  3. In Site Manager: Go to Site Settings > Manage Sites > Edit and add IFrame parameters for URL, Width and Height. Add the optional parameter profile_id=%ProfileID% to IFrame URL in 1) and 2) when the "Use Profile Id In URL" check box is selected and there is an available parameter value from webHotel pages. This profile Id will be the NAME_ID of a logged in user from webHotel using the Sign In functionality.
  4. In Site Manager: Go to Site Settings > Manage Sites > Edit > Display and set the parameter "Display quick search on home page" to Off.
