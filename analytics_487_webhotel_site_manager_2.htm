Add Analytics Tracking Code to the Site

There are 3 ways to add Analytics to the site:

  1. Add Analytics to the entire site
  2. Add Analytics to a specific page
  3. Add In-Page Analytics to track dynamic page events See Also: Adding In-Page Analytics to Track Specific Page Events

To add analytics to the entire site or to a specific page access the Analytics tab go to Site Settings > Manage Sites > Backend Settings > then select the Analytics tab.

The Analytics tab allows you to turn on and off the connection to external, third-party analytic providers.To add a new provider, select the Add Provider button and enter the provider's name. After a provider is added, you can enter scripts from the provider to set up analytics and tracking support for all pages or for specific pages.

Analytics Provider: This displays the name of the provider of the analytics software.

Configuration: Select the "On" radio button to turn on the connection to this provider. Select the "Off" radio button to turn off the connection to the provider.

Edit. Select the edit icon under this column to add scripts from the provider.

Adding Analytic Support from a Provider

To add analytics :

  1. Under the Edit column, select the Edit icon for the provider.
  2. On the Provider Details screen, select a page to edit from the Select a Page Type drop-down list.

  3. Select the Script tab and paste the script supplied by your provider into the field below.

  4. Select the Save Details button to save changes to the provider details.

  5. Repeat the above steps for each page you want to add analytic support.

Adding third-party analytics and tracker support for mobile sites

Site Manager supports third-party analytics and trackers for mobile sites. Each mobile site can use analysis and tracking scripts from an analytics provider to execute a predefined javascript code with populated variables on each step of the mobile booking path. For the below steps, Google will be used as the analytics provider.

Note: You must have a mobile analytics provider account before proceeding.

  1. Login to your Google Analytics account.
  2. Go to the Account Administration area of your Google account and add a new account for your mobile site.
  3. For the Site URL, enter the URL for your site (locate this under Site Manager>Site Settings) and add the necessary details to complete the account.
  4. Accept the terms of service agreement for the new account.
  5. Use single domain tracking and copy the provided tracking code script. The tracking ID for the account will automatically be added.
  6. In Site Manager > Manage Sites>Back End>Analytics, edit the Google Analytics Provider and paste the tracking script copied from the Google Account into the Script tab area for all page types. See the above procedure, "Adding Analytic Support from a Provider," for details on how to do this.

Note. Some pages load using AJAX so you may not be able to see the specific analytic's code for that page when you view the source code of the browser.
