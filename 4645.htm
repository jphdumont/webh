Change My Account Password Component (myRestaurant)

Add the Change Password Component to the myAccount.jsp Page Template to enable the ability for a logged in user to change their password from their my account page.

Frontend Example


To Add the Change Password Component to the myAccount.jsp Page Template:

  1. In Site Manager go to the Content tab and select Sections & Pages from the left navigation.
  2. Select Edit from next to Web Account Pages and select Edit next to the myAccount.jsp
  3. Then add the component Change Password to the page layout.
  4. Select Edit and Edit Page Layout to add the Change Password Component to the page layout.
