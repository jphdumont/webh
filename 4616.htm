Configure See Nearby Locations on the LocationDetails.jsp Page

Use the See Nearby Locations component to display the Nearby Stores.

Note. Dependant on data configured in myCentral.

Steps to Configure the See Nearby Locations

  1. In Site Manager go to the Content tab.
  2. Select the Site from the Edit Site drop down list.
  3. Select Sections & Pages from the left navigation, select Edit next to myRestaurant Web Online Ordering Pages.
  4. Select Edit next to the Location Details page LocationDetails.jsp page.

  1. Edit the page layout and add the See Nearby Locations component to the page layout.
