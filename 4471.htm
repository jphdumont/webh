Mobile Redirection

When the user accesses a site using a mobile device, the user could be redirected to another siite if this parameter is turned on.
To turn on and off the site mobile (site) redirection:

  1. In Site Manager go to Site Settings.
  2. Select Manage Site from the left navigation.
  3. Select the site's Backend settings and select the Mobile Redirection tab.
  4. Turn off to keep the user on the site. Or turn on to redirect the user to another site that is optimized for the mobile device. When the parameter is set to on select the site from the drop down list to redirect the when the user is using a mobile device.
