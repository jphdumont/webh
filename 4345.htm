Customize the Forgot Password Emails and Related Screen Labels (webHotel)

To customize the Email Templates:

  1. In Site Manager go to the Content tab and select the site from the Edit Site list.
  2. Select Email Templates from the left navigation.
  3. Select Edit next to the templates to edit.

Customize the Forgot Password Email

When the Site Manager Users requests a new password the user is sent a link to reset the password. To configure the the reset password email go to Content > Email Templates > Forget Password and use the new variable ${password-reset-url} in the email template. Once the password is reset the user returns to a new login page. The link expires once the password is reset or after 24 hours.

To customize the Screen Labels for the forgot password information messages:

  1. In Site Manager go to the Content Tab, and select and Edit Site from the drop down list.
  2. Select Screen Labels from the left navigation.
  3. From the drop down list Select a group go to the My Restaurant Web Information Messages.
  4. Search for the messages related to Password.
  5. Once you enter the text you MUST click outside the cell. And then select Apply and Save for the change to save.

Note. For changes to take effect, clear all cache from your site and the web browser before viewing the frontend of the site.
