Adding a Custom Menu Display for a Mobile Site's Home Page

Note. This is only applies to a mobile website in Site Manager.

A customized menu display can be configured for a Mobile Site's home page. The menu display can be customized with large icons that can each be linked to an internal page on the mobile site or to an external site, document or image file. The following button links are available for customizing the home page menu:

  • customBtnLink1
  • customBtnLink2
  • customBtnLink3
  • customBtnLink4

To enable and configure these button links, complete the below steps in Site Manager.

Step 1: Enabling the Custom Button Links

To enable and configure the custom button links:

  1. In Site Manager, go to the Content tab and select a mobile site from the Edit Site drop-down list.
  2. From the left-side panel, select Sections & Pages.
  3. Under Section & Pages, select the Edit icon for the "General Pages (Mobile)" site section. The "Pages" tab will load.
  4. Under the Pages tab, select the Components configuration icon for the "Home Page." The page will refresh and the "Components" grid will display for the Home Page.
  5. On the Components grid, select the Comp Id column header to sort the components.
  6. Locate the following components and enable rendering for them by clicking in the cell under the "Render" column.

    msm.homePage.customBtnLink1

    msm.homePage.customBtnLink2

    msm.homePage.customBtnLink3

    msm.homePage.customBtnLink4

  7. Under the Navigation column, enter a URL for each component to link to, use an internal page path or an external URL or file to redirect the user to when the custom button is clicked.
  8. Select the Save button to apply the configuration changes.

Step 2: Configure Custom Labels for the Button Links

To configure the custom labels for the menu buttons:

  1. In Site Manager, go to the Content tab and select a mobile site from the Edit Site drop-down list.
  2. From the left-side panel, select Screen Labels. The "Screen Label"s tab will load with the "Translation" grid displayed.
  3. Under the Screen Labels tab, select the Home (MSM) group from the "Select a group" drop-down list and locate the following label components:

    msm.homePage.customBtnLink1

    msm.homePage.customBtnLink2

    msm.homePage.customBtnLink3

    msm.homePage.customBtnLink4

  4. Locate the language in which you want to configure the label and enter the text for the label under the column.

Note: You can configure the labels in any language the site supports. However, the label components may not be found for the Home (MSM) group if you have not first accessed the actual site Home page after enabling the custom button links. For more information, see "Step 1: Enabling the Custom Button Links" above.

Step 3: Configuring Custom Icons for the Button Links

To configure the custom icons for the menu button links:

  1. In Site Manager, go Motif Management and select a mobile site from the Edit Site drop-down list.
  2. From the left-side panel, select Application Images. The "Custom Images" tab will load.
  3. Under "Image Categories," select Buttons.
  4. Under the "Group Name" drop-down list, select Home Page / Menu Dialog Button Icons group name and locate the following CSS tags:

    msm-homePage-customBtnLink1

    msm-homePage-customBtnLink2

    msm-homePage-customBtnLink3

    msm-homePage-customBtnLink4

  5. Select the Edit icon for each CSS Tag name and assign a button icon to it.The "Assign Image" dialog will open.
  6. From the Assign Image dialog, click the Select Image button. The "Manage Media" will open.
  7. From the Media Browser, double click on the image to be used for the button icon.
  8. Select the Apply button to add the image.
  9. Select the Save button to save and apply all changes to the CSS tags.
